# 1. 생성형 AI를 통한 제주 관광 데이터 분석

생성형 AI를 통하면 기존에 데이터분석가만 가능했었던 데이터 분석을 할 수 있습니다. colab에 gemini, chatGPT 등을 활용하여 제주 관광 데이터 분석을 해보세요. 이 챕터에서는 분석하는 방법까지 진행하진 않으며, 데이터가 있는 곳만 제공합니다. 실습형 수업이 있을 경우 해당 데이터를 분석하는 실무까지 진행합니다.

## 1.1 제주 관광 데이터 종류

```
제주 관광 데이터
├── 관광지 정보
│   ├── 명소 목록 (위치, 설명, 카테고리)
│   ├── 입장객 수 통계
│   └── 평점 및 리뷰
├── 숙박 정보
│   ├── 호텔/펜션/게스트하우스
│   ├── 가격대
│   └── 예약률
├── 음식점 정보
│   ├── 레스토랑/카페
│   ├── 메뉴 및 가격
│   └── 평점
├── 방문객 통계
│   ├── 월별/계절별 방문자 수
│   ├── 국적별 분포
│   └── 연령대별 분포
└── 교통 정보
    ├── 렌터카 이용률
    ├── 대중교통 이용 현황
    └── 주요 동선
```

## 1.2 데이터 수집 방법

**1. 공공데이터포털 (data.go.kr)**
```
- 제주특별자치도 관광정보
- 문화체육관광부 관광통계
- 한국관광공사 Tour API
```

**2. 제주관광공사 및 제주관광 빅데이터서비스 플랫폼**
```
- 제주 방문자 통계
- 관광 트렌드 리포트
- 주요 관광지 현황
```

**3. 기타 데이터 소스**
```
- 소셜 미디어 (인스타그램, 블로그)
- 리뷰 사이트 (트립어드바이저, 네이버 플레이스)
- 뉴스 및 언론 보도
```
