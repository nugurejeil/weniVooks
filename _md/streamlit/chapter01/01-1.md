# 1. Streamlit 개념

## **1.1 Streamlit 정의**

Streamlit은 **Python 기반의 오픈 소스 라이브러리**로, 사용자 친화적인 인터페이스를 통해 간편한 API를 제공하여 인터랙티브한 데이터 애플리케이션을 개발할 수 있도록 도와줍니다.

![image.png](/images/streamlit/chapter01/image.png)

공식 문서에서는 Streamlit을 “The fastest way to build and share data apps” 라고 정의하고 있습니다. 이는 데이터 애플리케이션을 구축하고 공유하는 가장 빠른 방법을 제공한다는 것을 의미합니다.

Streamlit을 사용하면 간단한 Python 스크립트를 작성하고 실행함으로써 데이터 시각화, 사용자 입력 폼, 테이블 등 다양한 기능을 가진 웹 애플리케이션을 빠르게 구축할 수 있습니다.

## **1.2 Streamlit 목표**

Streamlit의 주요 목표는 개발자, 데이터 분석가, PM, 마케터 등 누구든지 쉽게 개발할 수 있는 경험을 제공하는 것입니다. Streamlit은 간단한 코드 작성, 자동 서버 실행, 실시간 수정 및 테스트, 호환성, 배포 및 공유의 편의성을 통해 사용자가 빠르게 데이터 애플리케이션을 개발하고 다른 사람들과 공유할 수 있도록 지원합니다.

## **1.3 Streamlit 의 필요성**

Streamlit은 Python을 사용하여 간편하게 인터랙티브한 데이터 애플리케이션을 개발할 수 있는 강력한 라이브러리입니다. 이 라이브러리는 Pandas, Matplotlib, Plotly, Seaborn 등과 같은 다양한 데이터 과학 및 시각화 라이브러리와 원활하게 통합됩니다. 이를 통해 데이터 처리, 시각화, 통계 분석 등 다양한 작업을 쉽고 효율적으로 수행할 수 있습니다.

Streamlit의 큰 장점 중 하나는 배포와 공유의 간편성입니다. 로컬 환경에서 애플리케이션을 실행하거나 클라우드 플랫폼에 배포하여 다른 사람들과 손쉽게 공유할 수 있습니다. 이를 통해 데이터 과학과 웹 개발에 대한 지식이 부족한 사람들도 쉽게 데이터 애플리케이션을 개발하고 공유할 수 있습니다.

요약하자면, Streamlit은 Python을 사용하여 인터랙티브한 데이터 애플리케이션을 빠르고 쉽게 구축할 수 있는 라이브러리입니다. 코드 작성의 간소화, 자동 서버 실행, 실시간 업데이트, 다양한 라이브러리와의 호환성, 배포와 공유의 편의성 등의 특징을 가지고 있습니다.

# 2 Streamlit 기능

## **2.1 Streamlit 제공 API**

Streamlit은 다양한 API를 제공해 사용자가 웹 애플리케이션을 빠르게 구축하고 상호작용할 수 있도록 지원합니다. 주요 API는 다음과 같습니다.

데이터 출력 API

```python
import streamlit as st

# 다양한 데이터 유형 출력
st.write()                    # 데이터 형식 자동 맞춤출력
magic

# HEADINGS & BODY
st.markdown()                 # markdown 문자열 출력
st.title()                    # 제목 출력
st.header()                   # 헤더 출력
st.subheader()                # 서브헤더 출력

# FORMATTED TEXT
st.caption()                  # 텍스트 작게 출력
st.code()                     # 강조 표시의 코드블록
st.divider()                  # 수평 규칙 표시
st.echo()                     # 실행 코드 블록과 코드 실행 결과 출력
st.latex()                    # LaTeX 수학식
st.text()                     # 포맷된 일반 텍스트 출력

# UTILITIES
st.help()                     # 도움말 출력
st.html()                     # html 렌더링 결과 출력
```

데이터 입력 API

```python
# 텍스트 입력(Text input elements)
st.text_input()                  # 텍스트 입력 필드 생성
st.text_area()                   # 텍스트 영역 생성

# 숫자 입력 요소(Numeric)
st.number_input()                # 숫자 입력 필드 생성

# 날짜/시간 입력 요소(Date/time)
st.date_input()                  # 날짜 입력 필드 생성
st.time_input()                  # 시간 입력 필드 생성
```

위젯 API

```python
# 버튼 요소
st.button()                      # 버튼 생성
st.download_button()             # 다운로드 버튼 생성
st.form_submit_button()          # 폼 제출 버튼 생성
st.link button()                 # 링크 버튼 생성
st.page_link()                   # 페이지 또는 외부 페이지 링크 버튼 생성

# 선택 요소(상호작용을 통한 동적 데이터 시각화)
st.checkbox()                    # 체크박스 생성
st.color_picker()                # 색상 선택기 생성
st.feedback()                    # 피드백 선택기 생성
st.multiselect()                 # 멀티 셀렉트 박스 생성
st.pills()                       # 다중 알약 위젯 생성
st.radio()                       # 라디오 버튼 생성
st.segmented_control()           # 세그먼트 제어 위젯 생성
st.select_slider()               # 선택 슬라이더 생성
st.slider()                      # 슬라이더 생성
st.selectbox()                   # 셀렉트 박스 생성
st.toggle()                      # 토글 생성
```

레이아웃 & 컨테이너 API

```python
st.column()                    # 열 생성 및 너비와 배치 조정
st.container()                 # 컨테이너 생성
st.empty()                     # 빈 공간 생성
st.form()                      # 폼 컨테이너 생성
st.popover()                   # 목록 컨테이너 생성
st.expander()                  # 확장 패널(Expander) 생성
st.sidebar()                   # 사이드바 컨테이너 생성
st.tabs()                      # 탭(Tabs) 생성
```

미디어 입출력 API

```python
# 이미지,비디오,오디오 출력
st.image()                       # 이미지 출력
st.logo()                        # 로고 출력
st.audio()                       # 오디오 파일 출력
st.video()                       # 동영상 파일 출력
st.audio_input()                 # 오디오 입력을 받을 수 있는 입력 상자 생성
st.camera_input()                # 카메라 입력을 받을 수 있는 입력 상자 생성
```

진행률 & 상태 API

```python
# callout 요소
st.success()                      # 성공 메시지 출력
st.info()                         # 정보 메시지 출력
st.warning()                      # 주의 메시지 출력
st.error()                        # 에러 메시지 출력
st.exception()                    # 예외 메시지 출력

# 애니메이션 상태 요소
st.progress()                     # 진행률 표시
st.spinner()                      # 로딩중 애니메이션
st.status()                       # 상태 표시
st.toast()                        # 알림 메시지 출력
st.balloons()                     # 풍선 애니메이션
st.snow()                         # 눈 애니메이션
```

파일 입출력 API

```python
st.file_uploader()               # 파일 업로더 생성
```

데이터베이스 출력 API

```python
st.dataframe()                # DataFrame 데이터 표시
st.data_editor()              # 데이터 편집기 생성
st.table()                    # 테이블 형태 데이터 표시
Pandas Styler                 # 데이터프레임 표 스타일 적용
st.column_config()            # 원하는 형태로 데이터 구성
st.metric()                   # 메트릭 형태 표시
st.json()                     # JSON 데이터 표시
```

시각화 API

```python
# 기본 차트 요소
st.area_chart()                # 영역 차트
st.bar_chart()                 # 막대 차트
st.line_chart()				   # 라인 차트
st.scatter()				   # 스캐터 플롯 차트
st.map()				       # 지도 시각화

# 고급 차트 요소
st.pyplot()                    # Maplotlib 차트
st.plotly_chart()              # Plotly 차트
st.altair_chart()              # Vega-Altair 차트
st.vega_lite_chart()           # Vega-Lite차트
st.bokeh_chart()               # Bokeh 차트
st.pydeck_chart()              # Pydeck차트
st.graphviz_chart()            # dagre-d3 차트
```

## **2.2 외부 라이브러리 지원**

**데이터 시각화**

- Matplotlib: 데이터 시각화를 위한 기본인 라이브러리
- Seaborn: 통계적 그래픽스를 위한 고급 데이터 시각화 라이브러리
- Plotly: 인터랙티브한 웹 기반의 그래프와 차트를 생성하는 라이브러리

**머신 러닝 및 딥러닝**

- TensorFlow: 딥러닝 모델 개발과 훈련을 위한 라이브러리
- PyTorch: 딥러닝 모델의 구축과 훈련을 위한 라이브러리
- scikit-learn: 머신 러닝 알고리즘과 도구를 제공하는 라이브러리

**데이터 처리 및 분석**

- Pandas: 데이터 조작 및 분석을 위한 라이브러리
- NumPy: 과학적 계산을 위한 다차원 배열과 함수를 제공하는 라이브러리

**자연어 처리**

- NLTK (Natural Language Toolkit): 자연어 처리를 위한 도구 모음
- SpaCy: 고급 자연어 처리를 위한 라이브러리

# 3. 장단점 및 타 프레임워크와 비교

## 3.1 Streamlit 장점

**간편한 사용법**

개발 지식이 뛰어나지 않아도 Python 문법만으로 구현이 가능하기 때문에 Python을 사용해 본 사람이라면 어렵지 않게 사용이 가능합니다. 또한 직관적이고 사용방법이 간단하기 때문에 사용자가 쉽게 개발할 수 있습니다.

**빠른 개발 속도**

Streamlit 코드를 수정과 동시에 결과를 확인하여 실시간 업데이트된 내용을 확인할 수 있습니다. 이와 같은 자동 업데이트 기능으로 개발 프로세스를 보다 효율적으로 관리할 수 있습니다.

**뛰어난 모델 연동성**

다양한 외부 라이브러리와 모델을 손쉽게 통합할 수 있는 기능을 제공합니다. TensorFlow, PyTorch, scikit-learn 등의 인기 있는 머신 러닝 및 딥 러닝 프레임워크와 연동하여 모델의 결과를 시각화하거나 예측 서비스를 구현할 수 있습니다.

**반응형**

다양한 반응형 구성요소를 제공하여 사용자와 상호작용할 수 있는 인터페이스를 구축할 수 있습니다. 슬라이더, 버튼, 체크박스 등의 컴포넌트를 사용해 사용자 입력을 받고, 이를 기반으로 결과를 업데이트합니다.

**다양한 데이터 시각화 기능**

다양한 데이터 시각화 라이브러리와 툴을 지원합니다. Matplotlib, Seaborn, Plotly 등의 라이브러리를 사용해 차트, 그래프, 맵 등을 생성하고 데이터를 시각적으로 탐색하고 분석할 수 있습니다.

**배포와 공유**

로컬 환경에서 실행하거나 클라우드 플랫폼에 배포해 애플리케이션을 다른 사용자와 공유할 수 있습니다. 필요에 따라 애플리케이션을 웹 호스팅 서비스에 배포할 수도 있습니다.

## 3.2 Streamlit 단점

**제한된 웹 개발 기능**

Streamlit은 작은 규모나 프로토 타이핑에 적합합니다. html 페이지에서 동작이 발생할 때마다 리로드 되는 원리도 작동하기 때문에 정교한 레이아웃 혹은 고성능 웹 애플리케이션에는 한계가 있습니다.

## 3.3 타 시각화 프레임워크와 비교

데이터 시각화 및 대시보드 제작에 특화된 다양한 도구들이 존재하며, 각 도구는 기술 스택, 사용 목적, 커스터마이징 가능성에 따라 차별화됩니다. 아래는 Streamlit과 대표적인 시각화 도구들의 비교입니다.
![one.png](/images/streamlit/chapter01/one.png)

**Dash by Plotly**

Dash는 Plotly 라이브러리를 기반으로 한 Python 시각화 프레임워크입니다. 오픈소스와 기업용 라이선스가 혼합되어 있으며, 복잡한 대시보드 구성과 정교한 UI 제어에 적합합니다. Streamlit보다 진입 장벽이 높지만, 레이아웃과 상호작용을 세밀하게 제어할 수 있어 고급 사용자에게 유리합니다. 반면 Streamlit은 간결한 코드로 빠르게 앱을 구성할 수 있어 초심자나 프로토타입 개발에 강점을 보입니다.

**Power BI Pro**

Power BI는 Microsoft 생태계와의 강력한 통합을 바탕으로, 직관적인 UI와 리포트 작성 기능을 제공합니다. 연간 $120의 상용 툴로, Office 365 및 Excel과의 연계가 뛰어나며 기업 보고서 및 비즈니스용 대시보드에 적합합니다. Streamlit과 비교해 프로그래밍 지식 없이도 활용할 수 있는 반면, 복잡한 알고리즘이나 사용자 정의 분석에는 제약이 있습니다.

**Tableau Creator**

Tableau는 드래그 앤 드롭 기반의 고급 시각화 기능과 강력한 분석 성능을 갖춘 BI 툴로, 연간 $840의 비용이 발생합니다. 대기업에서 정형화된 리포트 및 대시보드 제작에 널리 활용되며, 학습 곡선이 낮아 분석 경험이 적은 사용자도 쉽게 접근할 수 있습니다. 하지만 Streamlit에 비해 사용자 정의나 머신러닝 모델 연동 등 고급 분석 기능에서는 유연성이 떨어질 수 있습니다.

**Looker Studio (구 Google Data Studio)**

Looker Studio는 Google 계정 기반으로 무료 제공되며, Google Analytics, BigQuery, Sheets 등과의 연동이 용이합니다. 마케팅 및 웹 트래픽 분석 대시보드 제작에 적합하며, 프로그래밍 없이 시각화를 구현할 수 있는 것이 장점입니다. Streamlit은 코드 기반의 유연한 커스터마이징이 가능해, 더 복잡한 분석이나 맞춤형 앱이 필요한 경우에 적합합니다.

**Observable**

Observable은 JavaScript 기반의 시각화 플랫폼으로, D3.js와의 자연스러운 통합을 통해 고급 인터랙티브 시각화를 지원합니다. 무료 및 프리미엄 요금제가 존재하며, 실시간 협업 기능이 강점입니다. JavaScript에 익숙한 사용자에게 적합하며, Streamlit은 Python 기반 분석가나 데이터 과학자에게 더 친화적인 도구입니다.
