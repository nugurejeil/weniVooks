# 1. 플로우 콘텐츠 (Flow Content)

<highlight>플로우 콘텐츠(Flow Content)는 대부분의 HTML 요소를 포함합니다.</highlight> 대부분의 웹 문서에서 사용하는 콘텐츠가 여기에 해당합니다.

| 예시 | `<div>`, `<p>`, `<a>`, `<table>`, `<header>`, `<footer>` 등 |
| --- | --- |
| 특징 | 거의 모든 HTML 요소가 플로우 콘텐츠에 속하며, 다른 콘텐츠 모델과 혼합될 수 있습니다. |

# 2. 대화형 콘텐츠 (Interactive Content)

<highlight>대화형 콘텐츠(Interactive Content)는 사용자가 상호작용할 수 있는 요소들을 포함합니다.</highlight>

| 예시 | `<a>`, `<button>`, `<input>`, `<select>`, `<textarea>`, `<details>`, `<label>` |
| --- | --- |
| 특징 | 사용자가 클릭, 입력, 선택 등 상호작용할 수 있는 요소들입니다. |

# 3. 구획 콘텐츠 (Sectioning Content)

<highlight>구획 콘텐츠(Sectioning Content)는 문서의 구조를 정의하고, 섹션을 나누는 데 사용됩니다.</highlight>

| 예시 | `<article>`, `<aside>`, `<nav>`, `<section>` |
| --- | --- |
| 특징 | 문서의 구획을 나누어 구조를 정의합니다. 문서의 아웃라인에 영향을 미칩니다. |

# 4. 제목 콘텐츠 (Heading Content)

<highlight>제목 콘텐츠(Heading Content)는 섹션의 제목을 정의합니다.</highlight>

| 예시 | `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>` |
| --- | --- |
| 특징 | 각 섹션의 제목을 나타내며, 문서의 구조를 이해하는 데 중요한 역할을 합니다. |

# 5. 구문 콘텐츠 (Phrasing Content)

<highlight>구문 콘텐츠(Phrasing Content)는 텍스트와 인라인 요소를 포함합니다.</highlight> 텍스트의 의미와 구조를 정의하는 데 사용됩니다.

| 예시 | `<span>`, `<a>`, `<em>`, `<strong>`, `<img>`, `<abbr>`, `<code>`, `<kbd>`, `<mark>` 등 |
| --- | --- |
| 특징 | 텍스트의 의미를 정의하고, 인라인 요소들로 구성됩니다. |

# 6. 내장 콘텐츠 (Embedded Content)

<highlight>내장 콘텐츠(Embedded Content)는 외부 소스의 콘텐츠를 포함합니다.</highlight>

| 예시 | `<img>`, `<audio>`, `<video>`, `<iframe>`, `<embed>`, `<object>` |
| --- | --- |
| 특징 | 외부 파일이나 리소스를 문서에 포함하는 요소들입니다. |

# 7. 메타데이터 콘텐츠 (Metadata Content)

<highlight>메타데이터 콘텐츠(Metadata Content)는 문서의 메타데이터를 정의하는 요소들입니다.</highlight>

| 예시 | `<base>`, `<link>`, `<meta>`, `<script>`, `<style>`, `<title>` |
| --- | --- |
| 특징 | 문서의 정보나 설정을 정의합니다. 보통 `<head>` 요소 내에 위치합니다. |
