## 1.3 유용한 단축키

개발 생산성을 높이기 위해 자주 사용하는 단축키를 익혀두면 좋습니다. Windows에서는 macOS의 Command 키가 Ctrl 키로, Option 키가 Alt 키로 대체됩니다.

| 기능 | macOS | Windows |
| --- | --- | --- |
| 탐색창 열기/닫기 | `Cmd + B` | `Ctrl + B` |
| 다른 이름으로 저장 | `Cmd + Shift + S` | `Ctrl + Shift + S` |
| 선택 줄 복사 | `Shift + Option + ↑/↓` | `Shift + Alt + ↑/↓` |
| 선택 줄 이동 | `Option + ↑/↓` | `Alt + ↑/↓` |
| 한 줄 삭제 | `Cmd + Shift + K` | `Ctrl + Shift + K` |
| 다중 라인 수정 | `Cmd + Option + ↑/↓` | `Ctrl + Alt + ↑/↓` |

더 많은 단축키는 공식 문서에서 확인할 수 있습니다.
