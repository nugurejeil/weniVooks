## 🔍 PART 1: 변수 완전 정복하기

### 지난번 계산기 코드 다시 보기

먼저 지난번 만든 계산기 코드 일부를 Chat에서 질문해보겠습니다.

**Chat에 입력:**

```
지난번에 만든 계산기에서 이 코드를 설명해주세요:

num1 = float(input("첫 번째 숫자: "))
num2 = float(input("두 번째 숫자: "))
result = num1 + num2
```

### 🤖 AI 선생님의 답변 예시

AI가 이런 식으로 답변할 겁니다:

```
이 코드에서 num1, num2, result는 모두 '변수'입니다.

- num1: 사용자가 입력한 첫 번째 숫자를 저장하는 변수
- num2: 사용자가 입력한 두 번째 숫자를 저장하는 변수  
- result: 계산 결과를 저장하는 변수

변수는 데이터를 저장하는 상자라고 생각하면 됩니다!
```

### 더 깊이 알아보기

**Chat에 계속 질문해보세요:**

```
변수에 대해 더 자세히 알고 싶어요. 
1. 변수 이름은 어떻게 정하나요?
2. 파이썬에서 변수 타입은 어떤 것들이 있나요?
3. 변수를 만들 때 주의할 점이 있나요?
```

### 🎯 실습 1: 변수 놀이터

**Chat에 이렇게 입력해보세요:**

```
변수 연습을 위한 간단한 코드를 만들어주세요.
다양한 타입의 변수를 만들고 출력하는 코드요.
```

AI가 이런 코드를 만들어줄 겁니다:

```python
# 다양한 변수 타입 연습
name = "김코딩"           # 문자열(str)
age = 25                  # 정수(int)  
height = 175.5            # 실수(float)
is_student = True         # 불린(bool)
hobbies = ["코딩", "독서", "음악"]  # 리스트(list)

print(f"이름: {name}")
print(f"나이: {age}세")
print(f"키: {height}cm")
print(f"학생 여부: {is_student}")
print(f"취미: {hobbies}")
```

**코드를 실행해보고 Chat에서 질문:**

```
이 코드에서 f-string이 뭐죠? 
그리고 각 변수 타입의 특징을 설명해주세요.
```

## 🔧 PART 2: 기본 함수 사용하기

### 우리가 이미 사용한 함수들

**Chat에 질문:**

```
print()와 input() 함수를 사용하는 방법을 
초보자도 이해할 수 있게 설명해주세요.
```

### 🤖 AI 선생님의 설명

```
print()와 input()은 파이썬에서 가장 기본적인 함수입니다:

- print(): 화면에 글자를 출력합니다
- input(): 사용자로부터 입력을 받습니다

예시:
print("안녕하세요!")  # 화면에 "안녕하세요!" 출력
name = input("이름을 입력하세요: ")  # 사용자 입력 받기
```

### 🎯 실습 2: print()와 input() 활용하기

**Chat에 이렇게 요청해보세요:**

```
print()와 input()을 사용해서 
사용자와 대화하는 간단한 프로그램을 만들어주세요.
```

AI가 만들어줄 코드:

```python
# 사용자와 대화하기
print("안녕하세요! 제가 AI 비서입니다.")
name = input("이름이 무엇인가요? ")

print(f"만나서 반가워요, {name}님!")

food = input("좋아하는 음식은 무엇인가요? ")
print(f"{food}를 좋아하시는군요! 저도 좋아해요!")
```

**실행 후 Chat에서 더 질문:**

```
1. f-string에서 f는 뭘 의미하나요? 
2. print() 안에 여러 줄을 출력하려면 어떻게 하나요?
3. input()으로 숫자를 받으려면 어떻게 하나요?
```

## 🎯 종합 실습: 배운 개념 활용하기

예상 결과:


### 간단한 계산기 만들어보기

```

간단한 계산기를 만들어주세요. 
두 숫자를 입력받아서 더하기만 하면 됩니다.
```

**예상 결과:**

```python
# 간단한 더하기 계산기
print("=== 간단한 계산기 ===")

num1 = input("첫 번째 숫자: ")
num2 = input("두 번째 숫자: ")

# 문자를 숫자로 바꾸기
number1 = float(num1)
number2 = float(num2)

# 계산하기
result = number1 + number2

# 결과 출력
print(f"{number1} + {number2} = {result}")
```

### 핵심 개념

```

✅ float(): 문자를 숫자로 바꾸는 함수
✅ f-string: 변수를 문자 안에 넣는 방법
```
