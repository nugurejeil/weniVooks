## 💡 오늘의 시작점

혹시 파이썬 코드를 보면서 이런 생각이 드셨나요?

```python

num2 = float(input("두 번째 숫자: "))

```

**이 코드를 보면서 떠오르는 궁금증들:**

- "이 `result = num1 + num2`에서 `result`는 정확히 뭐지?"

### 🎨 결과물 예시

AI가 만들어줄 완전한 프로그램:

```python
def calculate_grade(score):
    """성적을 받아서 학점을 반환하는 함수"""
    if score >= 90:
        return 'A'
    elif score >= 80:
        return 'B'
    elif score >= 70:
        return 'C'
    elif score >= 60:
        return 'D'
    else:
        return 'F'

def check_age_group(age):
    """나이에 따른 연령대 판별 함수"""
    if age < 13:
        return "어린이"
    elif age < 20:
        return "청소년"
    elif age < 65:
        return "성인"
    else:
        return "시니어"

def main():
    """메인 프로그램"""
    print("=== 학생 정보 관리 프로그램 ===")
    
    # 학생 정보 입력
    name = input("학생 이름: ")
    age = int(input("나이: "))
    score = float(input("성적: "))
    
    # 학점 계산
    grade = calculate_grade(score)
    
    # 연령대 판별
    age_group = check_age_group(age)
    
    # 결과 출력
    print(f"\n=== {name} 학생 정보 ===")
    print(f"나이: {age}세 ({age_group})")
    print(f"성적: {score}점")
    print(f"학점: {grade}")
    
    # 성적에 따른 메시지
    if grade in ['A', 'B']:
        print("우수한 성적입니다! 🎉")
    elif grade == 'C':
        print("보통 성적입니다. 조금 더 노력해보세요! 💪")
    else:
        print("더 많은 노력이 필요합니다. 화이팅! 🔥")

# 프로그램 실행
if __name__ == "__main__":
    main()
```

**구체적인 질문:**

```
❌ "함수가 뭐예요?"
✅ "함수의 매개변수가 어떻게 작동하는지 간단한 예시로 설명해주세요."
```

**맥락 제공:**

```
❌ "이 코드 수정해주세요."
✅ "이 코드에서 입력 검증 기능을 추가해주세요. 숫자가 아닌 값이 입력되면 다시 입력받도록 해주세요."
```

### 학습 효과 극대화

**복습 요청:**

```
"오늘 배운 변수, 함수, 조건문의 핵심 개념을 
표로 정리해서 요약해주세요."
```

**연결 학습:**

```
"오늘 배운 내용이 실제 개발에서 어떻게 활용되는지 
실무 예시로 설명해주세요."
```
