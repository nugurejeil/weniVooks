### 🎮 숫자 맞추기 게임 만들기

**Chat에 요청:**

```
Chat에 입력:
"1부터 10까지 숫자 맞추기 게임을 만들어주세요"
```

**AI가 생성한 코드:**

```python
import random

# 1부터 10 사이의 랜덤 숫자 생성
secret_number = random.randint(1, 10)
attempts = 0

print("=== 숫자 맞추기 게임 ===")
print("1부터 10 사이의 숫자를 맞춰보세요!")

while True:
    guess = int(input("숫자를 입력하세요: "))
    attempts += 1
    
    if guess < secret_number:
        print("더 큰 수를 입력하세요!")
    elif guess > secret_number:
        print("더 작은 수를 입력하세요!")
    else:
        print(f"정답입니다! {attempts}번 만에 맞추셨네요!")
        break
```

## 🌟 3단계: 완성된 프로그램 예시

### 할일 목록 관리 프로그램

**Chat에 요청:**

```
Chat에 입력:
"할일을 추가하고 볼 수 있는 간단한 프로그램을 만들어주세요"
```

**AI가 생성한 프로그램:**

```python
# 할일 목록 프로그램
todo_list = []

while True:
    print("\n=== 할일 목록 관리 ===")
    print("1. 할일 추가")
    print("2. 할일 목록 보기")
    print("3. 종료")
    
    choice = input("선택하세요: ")
    
    if choice == '1':
        task = input("할일을 입력하세요: ")
        todo_list.append(task)
        print(f"'{task}'가 추가되었습니다!")
    
    elif choice == '2':
        if todo_list:
            print("\n현재 할일 목록:")
            for i, task in enumerate(todo_list, 1):
                print(f"{i}. {task}")
        else:
            print("할일이 없습니다!")
    
    elif choice == '3':
        print("프로그램을 종료합니다.")
        break
    
    else:
        print("잘못된 선택입니다!")
```
