# 시작하기 전 준비사항 - 파이썬 설치와 터미널 기초

## 🎯 이번 섹션에서 준비할 것

- 파이썬이 설치되어 있는지 확인하고 설치하기
- 터미널(명령 프롬프트) 사용법 익히기
- 폴더와 파일 관리 기초
- Cursor에서 터미널 사용하는 방법

## 🤔 왜 이런 준비가 필요한가요?

**코딩을 시작하기 전에 기본 도구들이 준비되어 있어야 해요!**

### 💡 요리에 비유하면

```
🍳 요리를 하려면:
- 가스레인지 (파이썬)
- 도마와 칼 (터미널)
- 재료 보관할 냉장고 (폴더)

🖥️ 코딩을 하려면:
- 파이썬 (프로그래밍 언어)
- 터미널 (명령어 입력하는 곳)
- 폴더 (파일 정리하는 곳)
```

## 🐍 1단계: 파이썬 설치 확인하기

### 파이썬이 이미 설치되어 있는지 확인하기

**Windows 사용자:**

```
1. 키보드에서 "Windows키 + R" 누르기
2. "cmd" 입력하고 엔터
3. 검은 화면(명령 프롬프트)이 나타남
4. "python --version" 입력하고 엔터
```

**Mac 사용자:**

```
1. Spotlight 검색 (Cmd + Space)
2. "Terminal" 입력하고 엔터
3. 터미널이 열리면 "python3 --version" 입력하고 엔터
```

### 🎉 성공적으로 설치된 경우

```
> python --version
Python 3.9.7

또는

> python3 --version  
Python 3.9.7
```

### 🚨 설치되지 않은 경우

```
> python --version
'python'은(는) 내부 또는 외부 명령, 실행할 수 있는 프로그램, 또는 배치 파일이 아닙니다.

또는

> python3 --version
command not found: python3
```

## 🔧 2단계: 파이썬 설치하기 (필요한 경우)

### Windows 파이썬 설치

**1. 파이썬 공식 홈페이지 방문:**

```
1. 웹 브라우저에서 python.org 접속
2. "Downloads" 클릭
3. "Download Python 3.x.x" 버튼 클릭 (가장 최신 버전)
```

**2. 설치 파일 실행:**

```
⚠️ 중요: 설치 시 "Add Python to PATH" 체크박스 반드시 체크!
1. 다운로드된 파일 실행
2. "Add Python to PATH" ✅ 체크
3. "Install Now" 클릭
4. 설치 완료까지 기다리기
```

**3. 설치 확인:**

```
1. 새로운 명령 프롬프트 열기 (기존 것 닫고 다시 열기)
2. "python --version" 입력
3. Python 3.x.x 출력되면 성공!
```

### Mac 파이썬 설치

**Mac은 보통 Python이 기본 설치되어 있습니다:**

```
1. "python3 --version" 먼저 확인
2. 없다면 python.org에서 Mac용 설치 파일 다운로드
3. .pkg 파일 실행하여 설치
```

## 💻 3단계: 터미널 사용법 기초

### 🤔 터미널이 뭔가요?

**터미널은 컴퓨터에게 명령어로 지시를 내리는 곳이에요.**

### 💡 쉬운 설명

```
🖱️ 평소: 마우스로 폴더 클릭, 파일 더블클릭
⌨️ 터미널: 글자로 명령어 입력

예시:
- 폴더 열기: 마우스로 더블클릭 vs "cd 폴더명" 입력
- 파일 실행: 더블클릭 vs "python 파일명.py" 입력
```

### Windows 터미널 열기 방법

**방법 1: 시작 메뉴**

```
1. 윈도우 시작 버튼 클릭
2. "cmd" 또는 "명령 프롬프트" 검색
3. 명령 프롬프트 클릭
```

**방법 2: 단축키**

```
1. Windows키 + R
2. "cmd" 입력 후 엔터
```

**방법 3: 폴더에서 바로 열기**

```
1. 원하는 폴더 열기
2. 주소창에 "cmd" 입력 후 엔터
3. 해당 폴더에서 터미널 열림
```

### Mac 터미널 열기 방법

**방법 1: Spotlight**

```
1. Cmd + Space
2. "Terminal" 입력
3. 엔터
```

**방법 2: 응용 프로그램**

```
1. Finder > 응용 프로그램
2. 유틸리티 폴더
3. Terminal 더블클릭
```

## 📁 4단계: 폴더와 파일 관리 기초

### 기본 터미널 명령어

**현재 위치 확인:**

```bash
# Windows
cd

# Mac/Linux  
pwd
```

**폴더 내용 보기:**

```bash
# Windows
dir

# Mac/Linux
ls
```

**폴더 이동:**

```bash
# 모든 운영체제 공통
cd 폴더명        # 하위 폴더로 이동
cd ..           # 상위 폴더로 이동  
cd /            # 최상위 폴더로 이동
```

**폴더 만들기:**

```bash
# 모든 운영체제 공통
mkdir 폴더명
```

### 🎮 실습: 코딩용 폴더 만들기

**단계별 실습:**

```bash
# 1. 바탕화면으로 이동 (Windows)
cd Desktop

# 1. 바탕화면으로 이동 (Mac)
cd ~/Desktop

# 2. 코딩 연습용 폴더 만들기
mkdir cursor-practice

# 3. 만든 폴더로 이동
cd cursor-practice

# 4. 현재 위치 확인
pwd  # Mac
cd   # Windows
```

## 🎯 5단계: Cursor에서 터미널 사용하기

### Cursor 내장 터미널 열기

**방법 1: 메뉴에서**

```
1. Cursor 실행
2. Terminal > New Terminal 클릭
3. 하단에 터미널 패널 나타남
```

**방법 2: 단축키**

```
Windows: Ctrl + `(백틱)
Mac: Cmd + `(백틱)

💡 백틱(`)은 키보드 왼쪽 위, ESC 아래에 있어요
```

**방법 3: 명령 팔레트**

```
1. Ctrl+Shift+P (Windows) 또는 Cmd+Shift+P (Mac)
2. "Terminal: Create New Terminal" 검색
3. 엔터
```

### 🎮 터미널 테스트하기

**Cursor 터미널에서 파이썬 확인:**

```bash
# 터미널에 입력
python --version

# 결과 예시
Python 3.9.7
```

**간단한 파이썬 코드 실행:**

```bash
# 1. 파이썬 대화형 모드 실행
python

# 2. 간단한 코드 입력
print("안녕하세요!")

# 3. 종료
exit()
```

## 🚨 자주 발생하는 문제와 해결법

### 문제 1: "python을 찾을 수 없습니다"

**원인:** 파이썬이 설치되지 않았거나 PATH 설정 문제

**해결법:**

```
1. 파이썬 재설치 (반드시 "Add to PATH" 체크)
2. 컴퓨터 재시작
3. 새로운 터미널 열기
```

### 문제 2: "터미널이 안 열려요"

**Windows:**

```
1. 윈도우 시작 메뉴에서 "cmd" 검색
2. 명령 프롬프트를 관리자 권한으로 실행
3. 바이러스 백신 프로그램 확인
```

**Mac:**

```
1. Spotlight에서 "Terminal" 정확히 입력
2. 시스템 환경설정 > 보안 및 개인정보 확인
```

### 문제 3: "폴더 이동이 안 돼요"

**해결법:**

```
1. 폴더명에 띄어쓰기가 있으면 따옴표 사용:
   cd "My Documents"

2. 경로를 정확히 확인:
   pwd (Mac) 또는 cd (Windows)로 현재 위치 확인

3. 폴더가 실제로 존재하는지 확인:
   ls (Mac) 또는 dir (Windows)로 내용 보기
```

## 💡 추가 팁

### 터미널 사용이 어려우시다면

```
✅ 걱정하지 마세요! 
- Cursor에서는 대부분의 작업을 마우스로 할 수 있어요
- 터미널은 프로그램 실행할 때만 사용하면 됩니다
- 천천히 익숙해지시면 돼요
```

### 유용한 단축키들

```
# 터미널에서
Ctrl + C: 실행 중인 프로그램 중지
↑ ↓ 화살표: 이전에 입력한 명령어 불러오기
Tab: 폴더/파일명 자동완성

# Cursor에서  
Ctrl + `: 터미널 열기/닫기
Ctrl + Shift + `: 새 터미널 열기
```
