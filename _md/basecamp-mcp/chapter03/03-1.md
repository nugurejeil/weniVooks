# 1. MCP의 연결 기능

MCP는 토글 버튼으로 기본 연결을 지원합니다. 다만 이 토글 기능을 사용해서 연결한 기능은 생각보다 정확하지 못합니다.

::img{src="/images/basecamp-mcp/03-1-1.png"}

예를 들어, 아래와 같이 프롬프트를 작성하면 정확하지 못한 답변을 생성하거나, 확인할 수 없다고 답변합니다.

::img{src="/images/basecamp-mcp/03-1-2.png"}

이정도의 MCP를 만드는 것은 어렵지 않기 때문에 우리는 앞서 해본 것처럼 직접 MCP를 만들어볼 필요가 있습니다. 또한 지금 Drive, Gmail, Calendar는 검색만 가능합니다. 이 기능은 읽기만 가능하며, 심지어 읽은 정보도 그리 정확하진 않습니다.

# 2. MCP의 통합 추가하기

클로드의 `통합 추가하기` 버튼을 통해 다양한 솔루션과 통합할 수 있습니다. 

::img{src="/images/basecamp-mcp/03-1-3.png"}

예를 들어, Zapier를 이용하면 google docs를 비롯한 8000개의 앱과 통합할 수 있습니다. Zapier를 통하 앱을 연결하는 영상은 아래 녹화해두었으니 참고하세요.
