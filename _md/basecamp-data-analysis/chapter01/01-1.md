# 3. Google Colab 사용하기

위니북스를 이용하지 않고 다른 플랫폼을 사용하고 싶다면 **Google Colab**을 이용할 수 있습니다. Google Colab은 클라우드 기반의 무료 서비스로, Python 코드를 실행하고 실험할 수 있는 강력한 도구입니다.

## 3.1 Google Colab의 특징

- **Google 계정**만 있으면 누구나 쉽게 접근 가능
- **GPU를 활용**하여 복잡한 연산도 빠르게 수행 가능
- 클라우드 기반으로 **별도 설치 불필요**
- Jupyter Notebook 형태로 **코드와 문서를 함께 작성** 가능
