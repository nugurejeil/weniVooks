## 1.3 실습 메뉴

:::figure
::img{width="600" alt="" src="/images/basecamp-sql/chatper01-1/Untitled2.png"}
::figcaption[실습 메뉴 창]
:::

SQL Statement 화면은 다음과 같이 구성되어 있습니다.

1. **SQL 편집 영역**(중앙 큰 화면): SQL 구문을 작성하는 공간입니다. 교안에 나오는 SQL 코드를 여기에 입력합니다.
2. **RUN SQL 버튼**(하단): 작성한 SQL 구문을 실행하여 결과를 확인할 수 있습니다. 실행 단축키는 `Shift + Enter`입니다. 구문이 잘못되면 오류가 발생하거나 이전 결과가 유지될 수 있습니다.
3. **CSV 업로드 버튼**(상단 폴더 아이콘): 본인이 가지고 있는 CSV 파일을 업로드하여 데이터를 불러올 수 있습니다.
4. **코드 복사 버튼**(문서 겹침 아이콘): 작성한 SQL 구문을 클립보드로 복사합니다.
5. **코드 초기화 버튼**(회전 화살표 아이콘): 입력된 코드를 초기 상태(`SELECT * FROM 상품`)로 되돌립니다.

:::div{.callout}

**CSV 파일 업로드 시 주의사항**

CSV 파일을 업로드할 때 인코딩이 올바르지 않으면 데이터가 깨져서 업로드되지 않을 수 있습니다. 이런 경우 엑셀에서 파일을 다시 저장하거나, 인코딩을 **UTF-8**로 변경해주세요. 샘플 데이터로 테스트를 원하시면 아래 타이타닉 생존자 데이터를 활용할 수 있습니다.

:::
