## 5.1 실습 개요

이번 실습에서는 실제 데이터를 ChatGPT/Claude를 활용하여 분석하고, 그 결과를 **전문적인 분석 보고서** 또는 **인터랙티브 대시보드**로 만들어봅니다.

### 5.1.1 학습 목표

```
✅ 실제 비즈니스 데이터를 AI로 분석할 수 있다
✅ 분석 결과를 경영진이 이해하기 쉬운 형태로 정리할 수 있다
✅ 데이터 시각화를 통해 인사이트를 효과적으로 전달할 수 있다
✅ 실행 가능한 액션 플랜을 도출할 수 있다
```

### 5.1.2 실습 선택지

두 가지 옵션 중 하나를 선택하여 진행하세요:

**옵션 A: 분석 보고서 작성**
- PowerPoint 또는 PDF 형식
- 경영진 보고용
- 텍스트와 그래프 중심
- 소요 시간: 2-3시간

**옵션 B: 대시보드 만들기**
- Looker Studio, Tableau, Power BI
- 실시간 모니터링용
- 인터랙티브 시각화
- 소요 시간: 3-4시간

## 5.2 준비물

### 5.2.1 필수 준비물

```
□ ChatGPT Plus 또는 Claude Pro 계정
```

```
- 카페 운영 데이터
→ 강사에게 요청하거나 다운로드
```

## 5.3 옵션 A: 분석 보고서 작성

### 5.3.1 단계 1: 데이터 업로드 및 탐색

**ChatGPT 프롬프트:**

```
너는 데이터 분석 전문가야.

다음 데이터를 분석해줘:
[CSV 파일 업로드]

먼저 데이터 개요를 알려줘:
1. 총 행(row) 수와 열(column) 수
2. 각 컬럼의 의미와 데이터 타입
3. 데이터 기간 (시작일~종료일)
4. 결측치나 이상치가 있는지
5. 데이터 품질 평가

표 형식으로 정리해줘.
```

**기대 출력:**

| 항목 | 내용 |
|------|------|
| 총 행 수 | 1,234 |
| 총 열 수 | 8 |
| 데이터 기간 | 2024-01-01 ~ 2024-06-30 |
| 결측치 | 주소 컬럼에 3% |
| 이상치 | 매출액에 음수 2건 발견 |

### 5.3.2 단계 2: 핵심 질문 설정

**비즈니스 상황에 맞는 질문 3-5개를 정의하세요.**

**예시 (온라인 쇼핑몰):**
```
Q1. 어느 달에 매출이 가장 높았나? 이유는?
Q2. 가장 잘 팔리는 제품 카테고리는?
Q3. 어떤 고객층(연령대/지역)이 많이 구매하나?
Q4. 재구매율은 얼마나 되나?
Q5. 다음 분기 집중해야 할 제품/고객은?
```

### 5.3.3 단계 3: 심층 분석

**ChatGPT 프롬프트:**

```
위 데이터를 바탕으로 다음 질문들에 답해줘:

Q1. 월별 매출 트렌드 분석
   - 월별 매출액 그래프 (꺾은선 그래프)
   - 전월 대비 증감률 표
   - 매출 변동 이유 추정

Q2. 제품 카테고리 분석
   - 카테고리별 매출 비중 (파이 차트)
   - 카테고리별 성장률
   - 베스트셀러 TOP 10

Q3. 고객 분석
   - 연령대별 구매 패턴
   - 지역별 매출 분포 (지도 또는 막대 그래프)
   - 신규 vs 재구매 고객 비율

Q4. 재구매 분석
   - 재구매율 계산
   - 재구매까지 평균 기간
   - 재구매가 높은 제품은?

Q5. 인사이트 및 전략
   - 발견한 인사이트 5가지
   - 다음 분기 액션 플랜 3가지
   - 위험 요소 및 대응 방안

각 질문마다 시각화를 포함해서 설명해줘.
Python matplotlib 코드도 제공해줘.
```

### 5.3.4 단계 4: 시각화 생성

**ChatGPT가 제공한 Python 코드를 실행하거나, ChatGPT에게 이미지 생성 요청:**

```
위 분석 결과를 시각화해줘.

다음 그래프를 만들어줘:
1. 월별 매출 추이 (꺾은선 그래프)
2. 카테고리별 매출 비중 (파이 차트)
3. 연령대별 구매 금액 (막대 그래프)
4. 지역별 매출 분포 (막대 그래프)

각 그래프는:
- 제목 명확히
- 축 레이블 포함
- 색상 구분 명확히
- 전문적인 스타일

PNG 이미지로 다운로드 가능하게 제공해줘.
```

### 5.3.5 단계 5: 보고서 작성

**ChatGPT 프롬프트:**

```
위 분석 내용을 경영진 보고용 PowerPoint 형식으로 정리해줘.

슬라이드 구성:

[슬라이드 1] 표지
- 제목: "2024년 상반기 매출 분석 보고서"
- 부제: "데이터 기반 하반기 전략 수립"
- 분석 기간
- 작성자, 작성일

[슬라이드 2] Executive Summary
- 핵심 발견 3가지 (숫자 포함)
- 권고 사항 요약
- 한 페이지로 전체 내용 요약

[슬라이드 3] 데이터 개요
- 분석 데이터 설명
- 데이터 기간 및 범위
- 분석 방법론

[슬라이드 4-6] 주요 발견
- 슬라이드 4: 월별 매출 트렌드
  - 그래프
  - 주요 발견점 3-4개 (불릿 포인트)
- 슬라이드 5: 제품 및 고객 분석
  - 그래프 2개
  - 인사이트
- 슬라이드 6: 재구매 및 고객 충성도
  - 그래프
  - 분석 내용

[슬라이드 7] 심층 인사이트
- So What? (이 데이터가 의미하는 것)
- 발견한 패턴 및 트렌드
- 놀라운 발견 또는 예상 밖의 결과

[슬라이드 8] 액션 플랜
- 즉시 실행 사항 (1개월 내)
- 단기 개선 사항 (3개월 내)
- 중장기 전략 (6개월~1년)
- 각 액션별 담당 부서 제안

[슬라이드 9] 다음 단계
- 추가 분석이 필요한 영역
- 데이터 수집 개선 방안
- 다음 보고 일정

[슬라이드 10] Q&A

각 슬라이드별로:
- 슬라이드 제목
- 주요 내용 (불릿 포인트)
- 포함할 그래프/표
- 발표 시 강조할 포인트

를 상세히 정리해줘.
```

### 5.3.6 단계 6: PowerPoint 작성

**ChatGPT의 내용을 바탕으로 PowerPoint 제작:**

**팁:**
```
1. 템플릿 사용
   - 깔끔한 전문 템플릿 선택
   - 회사 CI 색상 활용

2. 그래프 삽입
   - ChatGPT가 생성한 이미지 다운로드
   - 또는 Excel로 데이터 입력 후 그래프 생성

3. 텍스트 최소화
   - 불릿 포인트는 5개 이하
   - 문장보다는 키워드
   - 숫자는 크고 명확하게

4. 일관성 유지
   - 폰트 2가지 이내
   - 색상 테마 통일
   - 레이아웃 일관성
```

### 5.3.7 완성 보고서 체크리스트

```
□ 표지에 제목, 날짜, 작성자 포함
□ Executive Summary로 핵심 요약
□ 모든 그래프에 제목과 축 레이블
□ 숫자는 구체적 (예: "약 10%" X → "12.3%" O)

□ 10-15분 내 발표 가능한 분량
```

## 5.4 옵션 B: 대시보드 만들기

### 5.4.1 대시보드란?

**대시보드의 특징:**
```
✅ 실시간 데이터 모니터링
✅ 인터랙티브 (클릭해서 세부 정보 확인)
✅ 자동 업데이트 (데이터 연동 시)
✅ 여러 차트를 한눈에
✅ 필터링 및 드릴다운 가능
```

**대시보드 vs 보고서:**

| 구분 | 보고서 | 대시보드 |
|------|--------|----------|
| 형식 | 정적 (PDF, PPT) | 동적 (웹) |
| 업데이트 | 수동 | 자동 가능 |
| 인터랙션 | 없음 | 있음 |
| 용도 | 일회성 분석 | 지속적 모니터링 |
| 난이도 | 쉬움 | 중간 |

### 5.4.2 도구 선택: Looker Studio (추천)

**Looker Studio를 추천하는 이유:**
```
✅ 완전 무료
✅ Google 계정만 있으면 OK
✅ 직관적인 인터페이스
✅ Google Sheets 연동 간편
✅ 공유 및 협업 쉬움
```

**접속:**
```
looker.google.com/studio
→ Google 계정으로 로그인
```

### 5.4.3 단계 1: 데이터 정제 및 Google Sheets 업로드

**ChatGPT로 데이터 정제:**

```
다음 데이터를 Looker Studio 대시보드용으로 정제해줘:
[CSV 파일 업로드]

요구사항:
1. 컬럼명을 영문으로 (예: "매출액" → "revenue")
2. 날짜 형식 통일 (YYYY-MM-DD)
3. 숫자 컬럼에서 특수문자 제거 (예: "1,000원" → 1000)
4. 카테고리는 일관된 이름으로
5. 결측치 처리

정제된 데이터를 CSV로 다운로드 가능하게 제공해줘.
```

**Google Sheets에 업로드:**
```
1. sheets.google.com 접속
2. 새 스프레드시트 생성
3. 파일 > 가져오기 > 업로드
4. ChatGPT에서 다운받은 CSV 선택
5. 데이터 확인
```

### 5.4.4 단계 2: Looker Studio 대시보드 생성

**1. 데이터 소스 연결:**
```
1. Looker Studio 접속
2. "만들기" > "데이터 소스"
3. "Google Sheets" 선택
4. 방금 업로드한 시트 선택
5. "연결" 클릭
```

**2. 보고서 만들기:**
```
1. "만들기" > "보고서"
2. 방금 연결한 데이터 소스 선택
3. "보고서에 추가"
```

### 5.4.5 단계 3: 차트 추가

**ChatGPT로 대시보드 레이아웃 설계:**

```
너는 데이터 시각화 전문가야.

다음 데이터를 기반으로 Looker Studio 대시보드 레이아웃을 설계해줘:
[데이터 개요 붙여넣기]

대시보드 목적: 온라인 쇼핑몰 매출 모니터링

다음을 포함해줘:
1. 상단 KPI 카드 4개 (큰 숫자로 표시)
   - 총 매출액
   - 주문 건수
   - 평균 주문액
   - 신규 고객 수

2. 메인 차트 영역 (6개)
   - 월별 매출 트렌드 (꺾은선 그래프)
   - 카테고리별 매출 (파이 차트)
   - 요일별 주문 패턴 (막대 그래프)
   - 지역별 매출 (지도 또는 막대)
   - TOP 10 제품 (테이블)
   - 신규 vs 재구매 비율 (도넛 차트)

3. 필터 영역
   - 날짜 범위 선택
   - 카테고리 필터
   - 지역 필터

레이아웃을 그림으로 그려서 보여주고,
각 차트의 설정 방법을 단계별로 알려줘.
```

**Looker Studio에서 차트 추가:**

**KPI 카드 만들기:**
```
1. 도구모음에서 "성과기록표" 클릭
2. 캔버스에 드래그하여 배치
3. 오른쪽 패널에서:
   - 측정항목: "매출액" 선택
   - 집계 방법: "합계"
   - 비교 기간: "이전 기간" (선택사항)
4. 스타일 탭에서 폰트 크기, 색상 조정
```

**꺾은선 그래프 만들기:**
```
1. "차트 추가" > "시계열 차트"
2. 캔버스에 배치
3. 설정:
   - 기간 측정기준: "날짜"
   - 측정항목: "매출액"
   - 정렬: 날짜 오름차순
4. 스타일:
   - 선 색상: 파란색
   - 데이터 포인트 표시
   - 격자선 추가
```

**파이 차트 만들기:**
```
1. "차트 추가" > "파이 차트"
2. 설정:
   - 측정기준: "카테고리"
   - 측정항목: "매출액"
   - 슬라이스 수: 10 (TOP 10)
3. 스타일:
   - 색상: 자동 또는 커스텀
   - 도넛 차트로 변경 (선택사항)
   - 라벨 표시
```

**테이블 만들기:**
```
1. "차트 추가" > "표"
2. 설정:
   - 측정기준: "제품명"
   - 측정항목: "매출액", "주문수"
   - 정렬: 매출액 내림차순
   - 행 수: 10
3. 스타일:
   - 헤더 색상
   - 교차 행 색상 (가독성)
   - 열 너비 조정
```

### 5.4.6 단계 4: 필터 추가

**날짜 필터:**
```
1. "추가" > "컨트롤" > "기간"
2. 캔버스 상단에 배치
3. 기본 기간: "지난 30일" 또는 "사용자 지정"
```

**카테고리 필터:**
```
1. "추가" > "컨트롤" > "드롭다운 목록"
2. 설정:
   - 필터 측정기준: "카테고리"
   - 다중 선택 허용
   - 기본값: "모두"
3. 위치: 상단 또는 좌측 사이드바
```

### 5.4.7 단계 5: 디자인 꾸미기

```
   ┌─────────────────────────────────┐

   │ KPI1 │ KPI2 │ KPI3 │ KPI4     │
   ├──────┴──────┴──────┴──────────┤
   │ 필터: 날짜 | 카테고리 | 지역    │
   ├───────────────┬─────────────────┤

3. 색상 일관성:
   - 메인 색상 1개 (예: 파란색)
```
