# 1. 생성형 AI의 분류

생성형 AI는 생성하는 콘텐츠 유형에 따라 분류할 수 있습니다.

```
생성형 AI
├── 텍스트 생성 (LLM: Large Language Model)
│   ├── ChatGPT, Claude, Gemini
│   └── 글쓰기, 번역, 코드, 대화 등
├── 이미지 생성
│   ├── Midjourney, DALL-E, Stable Diffusion
│   └── 일러스트, 사진, 디자인 등
├── 영상 생성
│   ├── Sora, Veo, Gemini
│   └── 단편 영상, 애니메이션 등
├── 음악 생성
│   ├── Suno, Udio
│   └── 작곡, 작사, 보컬 등
├── 음성 생성 (TTS)
│   ├── 클로바더빙, 타입캐스트, 파파고
│   └── 나레이션, 성우 등
└── 코드 생성
    ├── GitHub Copilot, Cursor, Claude Code
    └── 프로그래밍 자동화
```

# 2. 텍스트 생성형 AI

- **ChatGPT**
    - **제작사**: OpenAI
    - **월 사용 요금**: 무료 / Plus $20 / Team $25(연간) 또는 $30(월간) / Pro $200
    - **주요 기능**: 대화형 텍스트 생성, 요약, 번역, 질문 응답, 창작 글쓰기 지원
    - **웹 검색**: 유료 버전에서 실시간 웹 검색 가능
    - **파일 출력**: 텍스트 파일, PPT, PDF 생성 가능
    - **압축파일**: 가능
    - **그래프 그리기**: 가능
- **Claude**
    - **제작사**: Anthropic
    - **월 사용 요금**: 무료 / Pro $20 / Team $25(최소 5명)
    - **주요 기능**: 대화형 AI, 글쓰기 및 문서 요약, 대규모 파일 분석, 코드 작성
    - **웹 검색**: Pro 버전에서 실시간 웹 검색 가능
    - **파일 출력**: 문서 분석 후 요약 제공
    - **압축파일**: 가능 (압축 파일 내용 분석 지원)
    - **그래프 그리기**: 가능

# 3. 이미지 생성형 AI

- **Midjourney**
    - **주요 기능**: 텍스트에서 이미지 생성
    - **월 사용 요금**: Basic $10 / Standard $30 / Pro $60 / Mega $120
- **DALL·E 3** (OpenAI)
    - **주요 기능**: 텍스트에서 이미지 생성
    - **월 사용 요금**: ChatGPT Plus 이상 구독 시 포함
    - **파일 출력**: 고해상도 이미지 다운로드 가능
- **Stable Diffusion**
      - **주요 기능**: 오픈소스 이미지 생성 모델
      - **월 사용 요금**: 무료 (오픈소스)
- **Nanobanana** (Google)
    - **주요 기능**: 텍스트에서 이미지 생성
    - **월 사용 요금**: Google AI Studio에서 무료 사용 가능


# 4. 비디오 생성형 AI

- **Sora2** (OpenAI)
    - **주요 기능**: 텍스트 기반 고품질 비디오 생성 (최대 1080p, 20초)
    - **월 사용 요금**: ChatGPT Plus, Team, Pro 구독 시 사용 가능
- **Veo3** (Google)
    - **주요 기능**: 텍스트에서 비디오 생성
    - **월 사용 요금**: Google AI Studio, Gemini에서 제한적 무료 사용 가능

# 5. 음악 생성형 AI

- **Suno**
    - **주요 기능**: AI 기반 음악 생성 (작곡, 작사, 보컬 포함)
    - **월 사용 요금**: 무료 (제한적) / Pro $10 / Premier $30
    - 위니브 사용사례: [DJ 소울곰의 AI 음악 Lo-Fi 플레이리스트(Suno AI)](https://youtu.be/G93eK_ZUrwc)
- **Udio**
    - **주요 기능**: AI 기반 음악 생성 및 편집
    - **월 사용 요금**: 무료 (제한적) / Standard $10 / Pro $30


# 6. 코딩 보조 AI

- **Claude Code**
    - **제작사**: Anthropic
    - **월 사용 요금**: Claude 구독별 제한사용
    - **주요 기능**: CLI 기반 코드 생성, 편집, 디버깅
- **Codex**
    - **제작사**: OpenAI
    - **월 사용 요금**: ChatGPT 구독별 제한사용
    - **주요 기능**: 코드 생성, 코드 설명, 코드 변환
- **gemini CLI**
    - **제작사**: Google DeepMind
    - **월 사용 요금**: gemini 구독별 제한사용
    - **주요 기능**: 코드 생성, 코드 리뷰, 디버깅 지원
- **GitHub Copilot**
    - **제작사**: GitHub (Microsoft)
    - **월 사용 요금**: 무료 / 개인 $10 / Business $19 / Enterprise $39
    - **주요 기능**: 코드 자동 완성, 코드 리뷰 보조, AI 채팅
- **Cursor**
    - **주요 기능**: AI 통합 코드 에디터, 코드 자동 생성 및 편집
    - **월 사용 요금**: 무료 (제한적) / Pro $20 / Business $40
