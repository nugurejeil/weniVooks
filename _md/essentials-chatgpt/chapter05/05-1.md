## 5.1 왜 회사 데이터 분석이 중요한가?

### 5.1.1 데이터 기반 의사결정의 시대

과거에는 경영진의 경험과 직관으로 의사결정을 했다면, 이제는 **데이터가 말하게** 해야 합니다.

**전통적 의사결정:**
```
경영진: "내 경험상 이 제품이 잘 팔릴 것 같은데?"
→ 결과: 성공률 50%
```

**데이터 기반 의사결정:**
```
분석팀: "지난 3년 데이터 분석 결과, 이 제품이 30-40대 여성에게 인기"
       "이 타겟층은 우리 고객의 45%를 차지하며 성장 중"
→ 결과: 성공률 80%+
```

### 5.1.2 생성형 AI가 바꾸는 데이터 분석

**과거: 전문 분석가만 가능**
```
데이터 분석 프로세스:
1. SQL로 데이터 추출 (개발자 협업 필요)
2. Python/R로 분석 (프로그래밍 스킬 필요)
3. Tableau/PowerBI로 시각화 (툴 학습 필요)
4. 인사이트 도출 및 보고서 작성
→ 소요 시간: 3-5일
```

**현재: 생성형 AI 활용**
```
데이터 분석 프로세스:
1. ChatGPT/Claude에 CSV 파일 업로드
2. 자연어로 질문 ("월별 매출 트렌드 보여줘")
3. AI가 자동 분석 및 시각화
4. AI가 인사이트 제안
→ 소요 시간: 30분
```

## 5.2 회사 사내 데이터 분석

### 5.2.1 분석 가능한 사내 데이터 종류

**1. 매출 데이터**
```
- 일별/월별 매출액
- 제품별/카테고리별 판매량
- 고객별 구매 금액
- 지역별 매출 분포
- 결제 수단별 분석
```

**2. 고객 데이터**
```
- 고객 인구통계학적 정보 (연령, 성별, 지역)
- 신규/재구매 고객 비율
- 고객 생애 가치 (LTV)
- 이탈 고객 분석
- 고객 세분화
```

**3. 운영 데이터**
```
- 재고 현황 및 회전율
- 배송 소요 시간
- 고객 문의 및 불만 데이터
- 반품/교환률
- 직원 생산성 데이터
```

**4. 마케팅 데이터**
```
- 캠페인별 ROI
- 채널별 전환율
- 웹사이트/앱 트래픽
- 이메일 오픈율/클릭률
- SNS 참여율
```

### 5.2.2 ChatGPT를 활용한 사내 데이터 분석 실전

#### 예제 1: 매출 데이터 분석

**시나리오:**
```
당신은 온라인 쇼핑몰의 마케팅 팀장입니다.
지난 1년 간의 매출 데이터를 분석하여 다음 분기 전략을 수립하려 합니다.
```

**데이터 샘플:**
```csv
날짜,제품명,카테고리,수량,단가,매출액,지역,연령대
2024-01-05,무선이어폰,전자제품,2,89000,178000,서울,30대
2024-01-05,스마트워치,전자제품,1,299000,299000,부산,20대
2024-01-06,러닝화,스포츠,1,129000,129000,서울,40대
2024-01-06,요가매트,스포츠,3,45000,135000,인천,30대
...
```

**ChatGPT 프롬프트:**

```
너는 데이터 분석 전문가야.

다음 1년간의 매출 데이터를 분석해줘:
[CSV 파일 업로드]

분석 요청 사항:

1. 전체 현황
   - 총 매출액과 주문 건수
   - 월별 매출 트렌드 (그래프 포함)
   - 전월 대비 성장률

2. 제품 분석
   - 카테고리별 매출 비중
   - 베스트셀러 TOP 10
   - 수익성이 높은 제품 (매출액 기준)

3. 고객 분석
   - 연령대별 구매 패턴
   - 지역별 매출 분포
   - 고객당 평균 구매 금액

4. 시계열 분석
   - 성수기/비수기 파악
   - 요일별 매출 패턴
   - 특이사항 (급증/급감) 이유 추정

5. 전략 제안
   - 다음 분기 집중해야 할 제품/고객층
   - 매출 증대를 위한 3가지 액션 아이템
   - 위험 요소 및 대응 방안

결과는 경영진 보고용으로 요약해주고,
주요 인사이트는 불릿 포인트로 정리해줘.
```

**기대 결과:**

ChatGPT가 다음을 제공합니다:
- 📊 월별 매출 그래프
- 📈 카테고리별 비중 파이 차트
- 📉 트렌드 분석
- 💡 실행 가능한 인사이트

**시나리오:**

```
SaaS 제품을 운영 중인데, 최근 고객 이탈률이 증가하고 있습니다.

```

**데이터 샘플:**
```csv
고객ID,가입일,해지일,플랜,월요금,로그인수,기능사용수,문의수,만족도
C001,2023-01-15,2024-03-20,Pro,50000,45,123,2,3.5
C002,2023-02-20,,Basic,20000,89,234,0,4.5
C003,2023-03-10,2024-01-15,Pro,50000,12,34,5,2.0
...
```

**ChatGPT 프롬프트:**

```
너는 고객 성공(Customer Success) 전문가야.

다음 고객 데이터를 분석해서 이탈 패턴을 찾아줘:
[CSV 파일 업로드]

분석 요청:

1. 이탈 고객 프로필
   - 이탈률 계산 (전체, 플랜별)
   - 이탈 고객의 공통 특징
   - 이탈까지 평균 기간

2. 이탈 위험 신호
   - 로그인 수와 이탈의 관계
   - 기능 사용 수와 이탈의 관계
   - 문의 수와 이탈의 관계
   - 만족도 점수와 이탈의 관계

3. 플랜별 비교
   - Basic vs Pro 이탈률 차이
   - 각 플랜의 이탈 주요 원인

4. 예측 모델
   - 현재 활성 고객 중 이탈 위험이 높은 고객 특징
   - 어떤 지표가 가장 중요한 예측 인자인지

5. 개선 방안
   - 이탈을 줄이기 위한 5가지 액션 플랜
   - 각 액션의 우선순위와 예상 효과

상관관계 분석과 시각화를 포함해줘.
```

## 5.3 동종 업종 분석

### 5.3.1 왜 동종 업종 분석이 필요한가?

**벤치마킹의 중요성:**
```
우리 회사 매출 성장률: 10%
→ 좋은 건가? 나쁜 건가?

동종 업계 평균 성장률: 15%
→ 우리는 평균보다 뒤처지고 있음
→ 개선 필요
```

**경쟁 우위 파악:**
```
우리 강점:
- 고객 만족도 4.5/5.0
- 배송 속도 1.2일

경쟁사 평균:
- 고객 만족도 4.0/5.0
- 배송 속도 2.5일

→ 우리의 차별화 포인트를 마케팅에 활용!
```

### 5.3.2 동종 업종 데이터 수집 방법

**1. 공개 데이터 소스**

```
정부 및 공공기관:
- 통계청 (kostat.go.kr)
- 공정거래위원회 (ftc.go.kr)
- 산업별 협회 보고서
- 한국은행 경제 통계

민간 리서치:
- 시장 조사 보고서
- 업계 동향 리포트
- 증권사 리서치 보고서
```

**2. 경쟁사 공개 정보**

```
상장사:
- 사업보고서 (dart.fss.or.kr)
- 분기 실적 발표
- IR 자료

비상장사:
- 회사 홈페이지 보도자료
- 채용 공고 (회사 성장세 파악)
- 뉴스 기사
```

**3. 웹 크롤링 및 소셜 리스닝**

```
공개 데이터:
- 경쟁사 웹사이트 (제품 가격, 프로모션)
- 리뷰 사이트 (고객 평가)
- SNS (브랜드 언급량, 감성 분석)
```

### 5.3.3 ChatGPT를 활용한 동종 업종 분석

#### 예제 1: 경쟁사 비교 분석

**시나리오:**
```
당신은 배달앱을 운영하고 있습니다.
주요 경쟁사 3개와 비교 분석을 하려 합니다.
```

**ChatGPT 프롬프트:**

```
너는 시장 분석 전문가야.

우리 회사와 경쟁사 3개의 비교 데이터:

| 지표 | 우리 | 배민 | 쿠팡이츠 | 요기요 |
|------|------|------|----------|--------|
| 월간 활성 사용자(만명) | 120 | 1500 | 800 | 600 |
| 평균 주문액(원) | 18000 | 16000 | 17000 | 15000 |
| 배달료(원) | 3000 | 3500 | 2500 | 3000 |
| 평균 배달시간(분) | 35 | 40 | 30 | 45 |
| 앱 평점 | 4.2 | 4.1 | 4.3 | 3.9 |
| 가맹점 수(만개) | 5 | 30 | 15 | 20 |
| 프로모션 빈도 | 주 2회 | 주 3회 | 매일 | 주 1회 |

다음을 분석해줘:

1. 경쟁 구도 파악
   - 시장 점유율 추정
   - 각 서비스의 포지셔닝
   - 우리의 시장 위치

2. 강점/약점 분석 (SWOT)
   - 우리의 강점 (경쟁사 대비)
   - 우리의 약점
   - 기회 요소
   - 위협 요소

3. 벤치마킹 포인트
   - 경쟁사가 우리보다 잘하는 것 TOP 3
   - 각각을 어떻게 개선할 수 있을지

4. 차별화 전략
   - 우리만의 강점을 살린 마케팅 메시지
   - 빠르게 따라잡아야 할 격차
   - 장기적 차별화 방향

5. 액션 플랜
   - 3개월 내 실행 가능한 개선 사항
   - 6개월 로드맵
   - 1년 비전

시각화와 구체적인 수치를 포함해서 정리해줘.
```

## 5.4 실전 데이터 분석 워크플로우

### 5.4.1 1단계: 목표 설정

**명확한 질문 정의:**
```
나쁜 질문: "데이터 분석 좀 해줘"
좋은 질문: "지난 분기 매출이 10% 감소했는데,
            어떤 제품/지역/고객층에서 떨어졌는지 알고 싶어"
```

### 5.4.2 2단계: 데이터 준비

**데이터 정리 체크리스트:**
```
□ CSV, Excel 등 AI가 읽을 수 있는 형식
□ 컬럼명이 명확 (한글/영문 일관성)
□ 날짜 형식 통일 (YYYY-MM-DD 권장)
□ 결측치 처리 (빈 셀 확인)
□ 중복 데이터 제거
□ 민감 정보 마스킹 (개인정보 보호)
```

**ChatGPT로 데이터 정리:**
```
프롬프트:
"다음 데이터를 정리해줘:
[데이터 붙여넣기]

정리 사항:
1. 날짜 형식 통일 (YYYY-MM-DD)
2. 결측치가 있는 행 표시
3. 중복된 행 제거
4. 숫자 컬럼에 문자가 섞인 것 수정

정리된 CSV를 다운로드 가능하게 제공해줘."
```

### 5.4.3 3단계: 분석 실행

**단계별 분석:**
```
1단계: 기초 통계
   - 평균, 중앙값, 최댓값, 최솟값
   - 데이터 분포 확인

2단계: 그룹별 비교
   - 카테고리별, 기간별, 지역별 비교
   - 차이가 큰 그룹 식별

3단계: 트렌드 분석
   - 시간에 따른 변화
   - 계절성, 주기성 파악

4단계: 상관관계 분석
   - 어떤 요인들이 서로 연관되어 있는지
   - 인과관계 추정

5단계: 예측
   - 미래 추세 예측
   - 시나리오별 시뮬레이션
```

### 5.4.4 4단계: 인사이트 도출

**ChatGPT 프롬프트:**
```
위 분석 결과를 바탕으로:

1. 가장 중요한 발견 3가지 (So What?)
2. 각 발견이 비즈니스에 미치는 영향
3. 즉시 실행 가능한 액션 아이템 5개
   (우선순위, 담당 부서, 예상 효과 포함)
4. 추가 분석이 필요한 부분

경영진이 3분 안에 이해할 수 있게 요약해줘.
```

### 5.4.5 5단계: 보고서 작성

**ChatGPT 프롬프트:**
```
위 분석 내용을 다음 형식의 보고서로 작성해줘:

1. Executive Summary (한 페이지 요약)
   - 배경 및 목적
   - 주요 발견 3가지
   - 권고 사항

2. 상세 분석
   - 데이터 개요
   - 분석 방법론
   - 주요 발견 (그래프 포함)

3. 결론 및 제안
   - 액션 플랜
   - 예상 효과
   - 타임라인

4. 부록
   - 세부 데이터 테이블
   - 분석 가정

PowerPoint 형식으로 슬라이드별로 정리해줘.
각 슬라이드의 제목과 주요 내용을 알려줘.
```

## 5.5 주의사항 및 팁

### 5.5.1 데이터 보안

**민감 정보 보호:**
```
⚠️ ChatGPT에 업로드하면 안 되는 데이터:
- 고객 개인정보 (이름, 전화번호, 이메일)
- 주민등록번호, 카드번호
- 사업 기밀 (미공개 신제품, 계약 조건)
- 직원 인사 정보

✅ 안전한 방법:
- 데이터 익명화 (고객ID를 무작위 번호로 변경)
- 마스킹 (전화번호 뒷자리 제거)
- 샘플링 (전체 데이터 대신 일부만)
- 사내 LLM 사용 (폐쇄망)
```

### 5.5.2 분석 결과 검증

**AI 결과를 맹신하지 말 것:**
```
□ 계산 결과를 수동으로 샘플 검증
□ 말이 되는지 상식적으로 판단
□ 이상한 결과는 데이터 오류일 가능성
□ 중요한 의사결정은 복수 검증
```

### 5.5.3 효과적인 프롬프트 작성 팁

**구체적인 요청:**
```
나쁨: "분석해줘"
좋음: "월별 트렌드를 꺾은선 그래프로 보여주고,
       전년 동월 대비 증감률을 표로 정리해줘"
```

**맥락 제공:**
```
"우리는 B2C 전자상거래 회사야.
 주요 고객은 20-40대 여성이고,
 주력 상품은 뷰티/패션이야.

 이 맥락에서 다음 데이터를 분석해줘..."
```

**단계별 요청:**
```
처음부터 모든 걸 한 번에 요청하지 말고:

1차: "이 데이터의 기초 통계를 알려줘"
2차: "카테고리별로 나눠서 비교해줘"
3차: "성장률이 높은 카테고리 3개를 자세히 분석해줘"
```
