## 1.1 실제 업무에서의 AI 활용

AI를 업무에 도입한다는 것은 단순히 도구를 하나 추가하는 것이 아닙니다. 업무 방식 자체를 재설계하는 과정입니다. 이 섹션에서는 실제 위니브에서 AI를 어떻게 활용하고 있는지 소개합니다.

## 1.2 위니브의 AI 도구 생태계

위니브에서 사용하고 있는 AI 도구들을 소개합니다. 굵은 글자로 된 것이 필수 도구입니다.

- 기본: **(전직군 필수) Claude**, **(전직군 필수) Claude Code,** ChatGPT
- 자료수집: Gemini, Perplexity, NotebookLM
- SW 제작 및 문서작성: **(개발자 필수) VSCode Copilot**, **(전직군 필수) Claude Code**
- 이미지, 영상 제작: **(디자이너 필수) Midjourney**, ChatGPT - Sora, Gemini - Veo3, Gemini - Nanobanana
- 포스터, 썸네일, 웹 페이지 제작: Canva, figma make

위니브에서는 업무에 필요한 AI 도구의 유료 결제를 전 직원에게 지원합니다. 최상의 버전을 사용할 것을 권장합니다. AI를 통한 자동화와 효율화의 결과를 생각한다면 결코 아까운 비용이 아닙니다.

사람의 능력은 10% 향상시키기가 매우 힘들지만, AI는 다릅니다. 빠른 속도로 성능이 향상되고 있습니다. **회사 입장에서는 커피 2 ~ 3잔의 가격으로 인건비는 같은데, 전체적인 효율은 20% ~ 30% 올리는 효과를 누릴 수 있습니다.** 도입하지 않을 이유가 없습니다.

## 1.3 주요 AI 도구 상세 소개

### 1.3.1 Claude

Anthropic에서 개발한 대화형 AI로, 긴 문맥 이해와 정확한 답변에 강점이 있습니다.

**주요 기능:**
- 200,000 토큰(약 15만 단어)의 긴 문맥 처리
- 복잡한 문서 분석 및 요약
- 안전하고 정확한 답변 생성
- 로컬 Claude 실행환경에서 커스텀 MCP (Model Context Protocol) 지원

:::div{.callout}
**로컬 Claude 실행환경에서 커스텀 MCP 소개 영상:** https://youtu.be/oM65SNNrvws
:::

### 1.3.2 Claude Code

터미널에서 직접 사용하는 AI 코딩 도우미로, 코드 작성부터 디버깅까지 전 과정을 지원합니다.

**주요 기능:**
- 자동 코드 생성 및 수정
- 버그 탐지 및 수정
- 프로젝트 전체 맥락 이해
- Git 등 다양한 SW 통합

**활용 예시:**
- 새로운 기능 구현
- 레거시 코드 리팩토링
- 테스트 코드 자동 생성

### 1.3.3 Gemini

Google에서 개발한 멀티모달 AI로, 텍스트, 이미지, 영상을 함께 처리할 수 있습니다.

**주요 기능:**
- Google 검색 연동으로 최신 정보 제공
- 이미지 분석 및 생성
- YouTube 영상 요약
- Veo3, Nanobanana 등 영상 생성 기능

**활용 예시:**
- 최신 트렌드 조사
- 이미지 기반 정보 검색
- 프레젠테이션 자료 준비

### 1.3.4 NotebookLM

Google의 AI 기반 연구 노트로, 자료를 업로드하면 AI가 분석하고 대화할 수 있게 해줍니다.

**주요 기능:**
- 문서 업로드 및 분석
- 자료 간 연결 및 인사이트 도출
- 자동 요약 및 질의응답
- 팟캐스트 형식으로 요약 생성

**활용 예시:**
- 학습 자료 정리
- 프로젝트 문서 관리
- 시장 조사 자료 분석

### 1.3.5 VSCode Copilot

GitHub과 Microsoft가 함께 개발한 AI 코딩 어시스턴트입니다.

**주요 기능:**
- 실시간 코드 자동완성
- 주석으로 코드 생성
- 다양한 언어 지원
- IDE 통합

**활용 예시:**
- 빠른 프로토타이핑
- 반복 코드 자동 생성
- 함수 구현 제안

### 1.3.6 Midjourney

최고 품질의 AI 이미지 생성 도구입니다.

**주요 기능:**
- 고품질 이미지 생성
- 다양한 스타일 지원
- 이미지 편집 및 변형
- 커뮤니티 갤러리

**활용 예시:**
- 콘셉트 아트 제작
- 마케팅 이미지 생성
- 프레젠테이션 비주얼

### 1.3.7 Figma Make

Figma의 AI 기반 디자인 자동화 도구입니다.

**주요 기능:**
- 자동 레이아웃 생성
- 디자인 시스템 적용
- 반복 작업 자동화

**활용 예시:**
- 랜딩 페이지 빠른 제작
- 프로토타입 생성
- 디자인 시스템 구축

### 1.3.8 Perplexity

AI 검색 엔진으로, 질문에 대한 답변을 출처와 함께 제공합니다.

**주요 기능:**
- 실시간 웹 검색
- 출처 명시
- 후속 질문 제안

**활용 예시:**
- 빠른 정보 조사
- 최신 뉴스 확인
- 학술 자료 검색

## 1.5 업무활용에 주의할 점
- 생성된 문서가 의사결정에 영향을 미치는가?
    - 할루시네이션을 면밀이 검토할 필요
- 문서에 개인정보가 들어가는가?
    - 해당 정보를 활용하지 않는다고 했지만, 여러분이 타인의 개인정보를 개인정보 활용가능성이 있는 다른 회사에 올리는 것 자체가 개인정보보호법 위반여지가 있을 수 있음
- 비밀 유지 의무에 위반되는가?
- 윤리적 측면에 위반되지 않는가? ["칭찬만 해줘"… 논문속에 'AI 비밀명령' 숨겨](https://www.donga.com/news/Inter/article/all/20250701/131913049/1)
- 이런 것을 잡아내는 LLM 또한 정확하지 않습니다. ["챗GPT 안 썼다. 억울하다"…AI 역설 | 연합뉴스](https://www.yna.co.kr/view/AKR20250401155500505)
- 요약해달라고 할 때에는 꼭 레퍼런스 체크를 해야 합니다.
    - 일반 텍스트는 거의 오류없이 요약합니다.
    - 한글, PDF 등의 파일은 종종 오류가 납니다. 이는 할루시네이션 문제가 아니라 파싱 문제입니다.
    - 텍스트로 긁어 올리는 것을 권합니다. 또는 프롬프트를 정교화 해서 원하는 수치를 일부만 얻을 수 있게 하세요.

## 1.6 정리

- 위니브는 **전 직군**이 AI 도구를 적극 활용
- 업무 특성에 맞는 **도구 조합**이 중요
- **유료 버전** 사용 권장 (생산성 대비 저렴한 투자)
- 보안과 검증을 항상 염두에 두기
