## 개요

:::div{.box}
Google Anti-Gravity 사용 시 주의해야 할 사항들을 정리했습니다.
:::

## 1. 시스템 요구사항 확인

:::div{.callout}
**⚠️ 중요**<br>

- **Mac**: macOS 12.0(Monterey) 이상 지원 (64비트만 가능)
- **Windows**: Windows 10/11 지원 (64비트만 가능)
- **Linux**: 주요 배포판 지원 (Ubuntu, Fedora, Debian 등) [glibc >= 2.28, glibcxx >= 3.4.25 (e.g. Ubuntu 20. Debian 10, Fedora 36, RHEL 8)]

:::

## 2. 보안 설정

### Mac 보안 설정

:::div{.callout}
**⚠️ Mac 사용자 필수**<br>
Mac 사용자는 보안 설정에서 앱 실행 허용이 필요합니다.

- 시스템 설정 > 개인 정보 보호 및 보안
- "확인되지 않은 개발자의 앱 허용" 옵션

:::

### Windows 보안 설정

:::div{.callout}
**⚠️ Windows 사용자 권장**<br>
Windows 사용자는 바이러스 백신 예외 설정을 고려하세요.

- Windows Defender 예외 추가
- 타사 바이러스 백신 예외 설정

:::

## 3. 정기적인 업데이트

### 업데이트 확인

:::div{.callout}
**💡 업데이트 권장**<br>
최신 기능 및 보안 패치를 위해 정기적으로 업데이트하세요.

- 자동 업데이트 활성화 권장
- 수동 업데이트 시 정기 확인

:::

## 4. 네트워크 환경

### 안정적인 연결

:::div{.callout}
**🌐 네트워크 권장**<br>
안정적인 인터넷 연결 환경에서 사용을 권장합니다.

- 유선 연결 권장 (Wi-Fi보다 안정적)
- 네트워크 품질 확인
- 대용량 데이터 전송 시 데이터 사용량 주의
- 모바일 핫스팟 사용 시 데이터 제한 확인

:::

## 5. 성능 최적화

:::div{.callout}
**⚡ 성능 최적화**<br>
최적의 성능을 위해 다음 사항을 고려하세요.

- 불필요한 프로그램 종료
- 백그라운드 프로세스 확인
- 메모리 사용량 모니터링
- 정기적인 캐시 정리
- 디스크 공간 확인

:::
