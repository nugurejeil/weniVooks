# Google Anti-Gravity 장단점

## 개요

:::div{.box}
Google Anti-Gravity의 주요 장점과 단점을 정리했습니다.
:::

## 장점

:::div{.callout}
**주요 장점**

- **무료 제공**: 기본 기능 무료(단, 제한된 사용량)
- **크로스 플랫폼**: Mac, Windows, Linux 지원
- **최신 AI 기술**: Gemini 3 Ultra/Pro/Nano 모델 지원, 지속적인 업데이트
- **통합 개발 환경**: 코드 편집, 디버깅, Git, 터미널이 통합된 IDE
- **강력한 코드 생성**: 자연어로 코드 생성, 실시간 자동 완성, 코드 리팩토링
- **협업 기능**: 실시간 협업 편집, 코드 리뷰, 팀 관리
- **Google 생태계 통합**: Google 계정, GCP, Google Drive 등 연동
- **확장성**: 확장 프로그램, API 통합, 커스터마이징 가능

:::

## 단점

:::div{.callout}
**주요 단점 및 제한사항**

- **프리뷰 버전**: 아직 나온지 얼마되지않은 서비스이기 때문에 이따금 버그 및 이슈 발생
- **인터넷 연결 필수**: 대부분 기능이 온라인 환경에서만 작동, 오프라인 사용 제한적
- **Google 계정 필요**: 사용을 위해 Google 계정 필수, 다른 인증 방식 제한적
- **초기 학습 곡선**: AI 개발 플랫폼 특성상 다양한 기능과 설정 이해 필요
- **시스템 리소스 사용**: 8GB RAM 권장(16GB 이상 권장), CPU 집약적 작업
- **API 사용량 제한**: 무료 플랜의 제한된 API 호출량, 할당량 초과 시 비용 발생 가능
- **서비스 의존성**: Google 서비스 종속, 정책 변경 시 영향 가능
- **커뮤니티**: 레퍼런스 문서가 별로 없음, 아직은 활성화되지 않은 커뮤니티 등

:::
