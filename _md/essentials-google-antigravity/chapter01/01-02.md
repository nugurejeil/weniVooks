## 개요

:::div{.box}
Google Anti-Gravity의 주요 장점과 단점을 정리했습니다.
:::

## 장점

:::div{.callout}
**주요 장점**

- **무료 제공**: 기본 플랫폼 무료 사용
- **크로스 플랫폼**: Mac, Windows, Linux 지원
- **AI 에이전트 기반**: Agent Manager를 통한 에이전트 생성 및 관리
- **통합 개발 환경**: Editor, Browser, Command가 통합된 환경
- **다중 작업 지원**: Tab을 통한 여러 작업 동시 진행
- **웹 브라우징 통합**: Browser 기능으로 웹 리서치 및 정보 수집
- **명령 실행**: Command를 통한 터미널 명령 및 스크립트 실행
- **자동 업데이트**: 정기적인 업데이트 알림 및 자동 업데이트
- **직관적인 인터페이스**: `Cmd/Ctrl + E` 등 간편한 단축키

:::

## 단점

:::div{.callout}
**주요 단점 및 제한사항**

- **Chrome Extension 필수**: Browser 기능 사용을 위해 Chrome Extension 설치 필수
- **인터넷 연결 필수**: Browser 기능 등 대부분 기능이 온라인 환경에서만 작동
- **Google 계정 필요**: 사용을 위해 Google 계정 필수
- **초기 학습 곡선**: Agent Manager, Editor, Browser, Command 등 다양한 기능 이해 필요
- **시스템 리소스 사용**: 여러 에이전트 동시 실행 시 리소스 사용 증가
- **호환성 문제**: Chrome Extension과의 호환성 확인 필요
