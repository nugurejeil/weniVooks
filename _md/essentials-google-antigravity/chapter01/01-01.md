## 소개

**Google Anti-Gravity**는 구글이 제공하는 무료 AI 개발 플랫폼입니다. 개발자들이 AI 에이전트를 활용하여 코드 개발, 웹 브라우징, 명령 실행 등을 수행할 수 있는 통합 개발 환경을 제공합니다.

![Google Anti-Gravity 로고](/images/essentials-google-antigravity/01-01-logo.png)

:::div{.box}
**💡 핵심 특징**

- 무료 제공 (기본 기능)
- 크로스 플랫폼 지원 (Mac, Windows, Linux)
- AI 에이전트 기반 작업 수행
- 통합 개발 환경 (Editor, Browser, Command)

:::

## 출시 배경

AI 기술의 급속한 발전과 함께 개발자들은 더욱 강력하고 사용하기 쉬운 AI 도구를 필요로 하게 되었습니다. Google은 이러한 요구에 부응하여 개발자 친화적인 AI 플랫폼을 제공하기 위해 Anti-Gravity를 개발했습니다.

## 플랫폼 특징

### 핵심 구성 요소

**Agent Manager**

- AI 에이전트 생성 및 관리
- 여러 에이전트 동시 실행
- 에이전트별 작업 히스토리 관리

**Editor**

- 코드 편집 기능
- 파일 생성 및 수정
- 코드 하이라이팅 및 자동 완성

**Browser**

- 웹 브라우징 기능
- 웹 페이지 탐색 및 상호작용
- Chrome Extension 연동 필요

**Command**

- 명령 실행 기능
- 터미널 명령 실행
- 스크립트 실행

**Tab**

- 탭 기반 인터페이스
- 여러 작업 동시 진행
- 탭 간 전환 및 관리

### 크로스 플랫폼 지원

- **Mac**: macOS 12 (Monterey) 이상
- **Windows**: Windows 10 (64비트) 이상
- **Linux**: glibc >= 2.28, glibcxx >= 3.4.25

## 주요 기능

### 1. AI 에이전트 관리

- Agent Manager를 통한 에이전트 생성
- 여러 에이전트 동시 실행 및 관리
- 에이전트별 작업 히스토리 추적

### 2. 코드 편집

- Editor를 통한 코드 작성 및 수정
- 파일 생성 및 편집
- 코드 하이라이팅 지원

### 3. 웹 브라우징

- Browser를 통한 웹 페이지 탐색
- Chrome Extension 연동 필수
- 웹 콘텐츠와의 상호작용

### 4. 명령 실행

- Command를 통한 터미널 명령 실행
- 스크립트 실행 지원
- 시스템 명령 수행

### 5. 탭 관리

- 여러 탭으로 작업 분리
- 탭 간 전환 및 관리
- 작업 컨텍스트 유지

## 사용 사례

### 개발 작업

- 코드 작성 및 편집 (Editor)
- 웹 리서치 및 정보 수집 (Browser)
- 명령 실행 및 자동화 (Command)
- 여러 작업 동시 진행 (Tab)

### AI 에이전트 활용

- 에이전트를 통한 자동화 작업
- 복잡한 작업의 단계별 수행
- 작업 히스토리 추적 및 관리
