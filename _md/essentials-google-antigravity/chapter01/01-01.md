## 소개

**Google Antigravity**는 에이전트 중심 개발 플랫폼입니다. 2025년 11월에 출시하였으며 IDE를 에이전트 퍼스트 시대로 진화시켜, 개발자가 작업(task) 단위로 에이전트를 관리할 수 있게 합니다. 에이전트는 에디터, 터미널, 브라우저를 자율적으로 오가며 기능 개발, UI 반복 작업, 버그 수정, 리서치, 보고서 생성 등 복잡한 소프트웨어 작업을 수행합니다.

![Google Anti-Gravity 로고](/images/essentials-google-antigravity/01-01-logo.png)

**💡 핵심 특징**

- 통합 개발 환경 (IDE)
- AI <-> 크롬 브라우저와의 연동을 통한 프론트엔드 개선

## 출시 배경

AI 기술의 급속한 발전과 함께 개발자들은 더욱 강력하고 사용하기 쉬운 AI 도구를 필요로 하게 되었습니다. Google은 이러한 요구에 부응하여 개발자 친화적인 AI 플랫폼을 제공하기 위해 Anti-Gravity를 개발했습니다.

## 플랫폼 특징

### 통합 개발 환경

- 코드 편집, 디버깅, 버전 관리가 통합된 환경
- AI 모델과의 상호작용을 위한 전용 인터페이스
- 프로젝트 관리 및 협업 도구

### AI 모델 지원

- Google의 최신 Gemini 모델 시리즈 지원
  - Gemini 3 Ultra: 최고 성능 모델
  - Gemini 3 Pro: 균형잡힌 성능
  - Gemini 3 Nano: 경량 모델
- 다양한 AI 작업 지원:
  - 코드 생성 및 자동 완성
  - 자연어 처리
  - 이미지 생성 및 분석
  - 데이터 분석

### 크로스 플랫폼 지원

- **Mac**: macOS 12.0(Monterey) 이상 지원 (64비트만 가능)
- **Windows**: Windows 10/11 지원 (64비트만 가능)
- **Linux**: 주요 배포판 지원 (Ubuntu, Fedora, Debian 등) [glibc >= 2.28, glibcxx >= 3.4.25 (e.g. Ubuntu 20. Debian 10, Fedora 36, RHEL 8)]

## 주요 기능

| 기능 | 설명 |
|------|------|
| **AI 기반 IDE** | Agent, Tab, Command 등 기존 AI 기능을 모두 지원하는 IDE |
| **비동기 에이전트** | 여러 워크스페이스에서 병렬로 작업하는 로컬 에이전트 |
| **Agent Manager** | 플래닝 모드, 대화 UI, Artifact 리뷰 중심의 에이전트 관리 화면 |
| **멀티 윈도우** | Editor, Manager, Browser 3개 창으로 구성된 제품 |
| **Browser Agent** | 대시보드 확인, SCM 작업, UI 테스트 등 브라우저 조작을 수행하는 에이전트 |

| 화면 | 설명 |

## 핵심 용어

| 용어 | 설명 |
|------|------|
| **Agent** | Antigravity의 핵심 AI 모달리티. Editor에서 밀접하게 작업하거나, Agent Manager로 여러 에이전트를 오케스트레이션 가능 |
| **Tab** | 강화된 자동완성 기능 |
| **Command** | 인라인 명령 입력 방식 |
| **Artifacts** | 에이전트가 작업 수행 또는 결과 전달을 위해 생성하는 산출물 (마크다운, diff 뷰, 아키텍처 다이어그램, 이미지, 브라우저 녹화 등) |

## 사용 사례

### 전문 개발자

Antigravity는 AI가 탑재된 코드 편집기입니다.
여기서 핵심은 **Agent Manager**입니다.

Agent Manager는 다음과 같은 일을 합니다.

- 밀린 작업들을 AI가 백그라운드에서 대신 처리해줍니다.
- 코드베이스를 분석해서 필요한 정보를 찾아줍니다.
- 이 창 저 창 왔다갔다 하는 수고를 줄여줍니다.

**쉽게 말해, AI 비서가 잡일을 대신 처리하는 동안 여러분은 중요한 작업에 집중할 수 있습니다.**

### 프런트엔드 개발자

**AI가 브라우저 화면을 직접 보면서 작업합니다.**

- 화면이 어떻게 보이는지 AI가 실시간으로 확인합니다.
- "버튼이 왼쪽으로 치우쳤네요" 같은 피드백을 스스로 파악합니다.
- 반복적인 수정 작업을 AI가 자동으로 처리합니다.

**결과적으로 예쁘고, 빠르고, 반응형인 웹페이지를 더 쉽게 만들 수 있습니다.**

### 풀스택 개발자

**검증**에 초점을 맞춥니다. AI가 코드를 짜고 끝이 아닙니다. 아래의 과정을 거칩니다.

1. AI가 작업 결과물(Artifact)을 보여줍니다.
2. 지금 무슨 작업 중인지 태스크 목록으로 알려줍니다.
3. 여러분이 확인하고 승인하면 다음 단계로 넘어갑니다.

**왜 이렇게 할까요? AI가 짠 코드를 믿을 수 있어야 복잡한 풀스택 프로젝트를 맡길 수 있기 때문입니다.**
