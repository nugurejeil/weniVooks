# Google Anti-Gravity 개요

:::div{.callout}
**학습 목표**

- Google Anti-Gravity의 주요 기능과 특징을 이해할 수 있다
- Gemini, Gemini CLI, Anti-Gravity의 차이점을 구분할 수 있다
- 플랫폼의 사용 사례와 장단점을 파악할 수 있다

:::

## 소개

Google Anti-Gravity는 구글이 제공하는 무료 AI 개발 플랫폼입니다. 2025년 11월에 출시되었으며, 개발자들이 AI 모델을 쉽게 활용하고 개발할 수 있도록 도와주는 통합 개발 환경(IDE)을 제공합니다.

![Google Anti-Gravity 로고](/images/essentials-google-antigravity/01-01-logo.png)

:::div{.box}
**💡 핵심 특징**

- 무료 제공 (기본 기능)
- 크로스 플랫폼 지원 (Mac, Windows, Linux)
- 최신 Gemini AI 모델 통합
- 통합 개발 환경 (IDE)

:::

## 출시 배경

AI 기술의 급속한 발전과 함께 개발자들은 더욱 강력하고 사용하기 쉬운 AI 도구를 필요로 하게 되었습니다. Google은 이러한 요구에 부응하여 개발자 친화적인 AI 플랫폼을 제공하기 위해 Anti-Gravity를 개발했습니다.

## 플랫폼 특징

### 통합 개발 환경

- 코드 편집, 디버깅, 버전 관리가 통합된 환경
- AI 모델과의 상호작용을 위한 전용 인터페이스
- 프로젝트 관리 및 협업 도구

### AI 모델 지원

- Google의 최신 Gemini 모델 시리즈 지원
  - Gemini 3 Ultra: 최고 성능 모델
  - Gemini 3 Pro: 균형잡힌 성능
  - Gemini 3 Nano: 경량 모델
- 다양한 AI 작업 지원:
  - 코드 생성 및 자동 완성
  - 자연어 처리
  - 이미지 생성 및 분석
  - 데이터 분석

### 크로스 플랫폼 지원

- **Mac**: macOS 12.0(Monterey) 이상 지원 (64비트만 가능)
- **Windows**: Windows 10/11 지원 (64비트만 가능)
- **Linux**: 주요 배포판 지원 (Ubuntu, Fedora, Debian 등) [glibc >= 2.28, glibcxx >= 3.4.25 (e.g. Ubuntu 20. Debian 10, Fedora 36, RHEL 8)]

## 주요 기능

### 1. AI 기반 코드 생성

- 자연어 설명으로 코드 자동 생성
- 코드 자동 완성 및 제안
- 코드 리팩토링 지원
- 버그 수정 제안

### 2. 통합 채팅 인터페이스

- AI와의 대화형 인터페이스
- 컨텍스트 인식 대화
- 멀티턴 대화 지원
- 대화 히스토리 관리

### 3. 프로젝트 관리

- Git 통합
- 프로젝트 템플릿
- 의존성 관리
- 빌드 및 배포 도구

### 4. 협업 기능

- 실시간 협업 편집
- 코드 리뷰 도구
- 팀 채팅 및 알림
- 공유 작업 공간

### 5. 확장성

- 확장 프로그램 지원
- 커스텀 플러그인 개발
- API 통합
- 웹훅 지원

## 사용 사례

### 소프트웨어 개발

- 웹 애플리케이션 개발
- 모바일 앱 개발
- 데이터 과학 프로젝트
- 머신러닝 모델 개발

### 교육 및 학습

- 프로그래밍 학습 도구
- 코드 예제 생성
- 개념 설명 및 튜토리얼

### 프로토타이핑

- 빠른 프로토타입 개발
- 아이디어 검증
- 개념 증명(POC) 개발

## Google AI 제품 비교

Google의 AI 관련 제품들 간의 차이점을 비교한 표입니다.


| 항목 | Gemini (웹/앱) | Gemini CLI | Anti-Gravity |
|------|----------------|------------|--------------|
| **제품 유형** | 웹 기반 AI 챗봇/앱 | 명령줄 인터페이스 | 통합 개발 환경 (IDE) |
| **주요 용도** | 일반적인 AI 대화, 질문 답변 | 스크립팅, 자동화, 배치 작업 | 소프트웨어 개발, 코드 생성 |
| **인터페이스** | 그래픽 사용자 인터페이스 (GUI) | 텍스트 기반 명령줄 | 데스크톱 애플리케이션 (GUI) |
| **코드 편집** | 제한적 (코드 블록 표시) | 코드 실행 및 스크립팅 | 전문 코드 편집기 통합 |
| **프로젝트 관리** | 없음 | 제한적 | Git 통합, 프로젝트 관리 |
| **디버깅** | 없음 | 기본 로깅 | 통합 디버거 |
| **버전 관리** | 없음 | 없음 | Git 통합 |
| **확장성** | 브라우저 확장 프로그램 | 셸 스크립트 통합 | 확장 프로그램, 플러그인 |
| **학습 곡선** | 낮음 (직관적) | 중간 (CLI 사용법 필요) | 중간-높음 (IDE 기능 학습) |
| **대화형 사용** | 높음 (채팅 중심) | 낮음 (명령 실행 중심) | 높음 (채팅 + 코드 편집) |
| **코드 생성** | 가능 (텍스트 출력) | 가능 (스크립트 실행) | 전문적 (컨텍스트 인식) |
| **코드 완성** | 없음 | 없음 | 실시간 자동 완성 |
| **리팩토링** | 수동 (요청 필요) | 수동 | 자동 제안 |
| **통합 개발 환경** | 아니오 | 아니오 | 예 (완전한 IDE) |
| **터미널 통합** | 없음 | 네이티브 | 통합 터미널 |
| **파일 시스템 접근** | 제한적 | 전체 접근 | 전체 접근 |
| **API 통합** | REST API | REST API, CLI | REST API, WebSocket, 확장 API |
| **오프라인 사용** | 불가능 | 제한적 (캐시된 응답) | 제한적 (기본 편집만) |
| **사용자 대상** | 일반 사용자, 비개발자 | 개발자, 시스템 관리자 | 소프트웨어 개발자 |
| **가격** | 무료 (기본), 유료 (고급) | 무료 | 무료 (기본), 유료 (출시예정) |
| **설치 필요** | 없음 (웹) 또는 앱 설치 | CLI 도구 설치 | 데스크톱 앱 설치 |
| **크로스 플랫폼** | 웹 브라우저, 모바일 앱 | Mac, Windows, Linux | Mac, Windows, Linux |
