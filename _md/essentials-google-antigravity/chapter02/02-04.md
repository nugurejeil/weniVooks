# 문제 해결

## 개요

:::div{.box}
Google Anti-Gravity 설치 및 실행 중 발생할 수 있는 일반적인 문제와 해결 방법을 안내합니다.
:::

## 일반적인 문제

### 설치 문제

#### Mac: "확인되지 않은 개발자" 오류

:::div{.callout}
**증상:**<br>
- 앱 실행 시 "확인되지 않은 개발자" 경고 메시지 표시<br>
- 앱이 실행되지 않음
:::

**해결 방법:**

1. **시스템 설정에서 허용**
   ```
   1. 시스템 설정 열기
   2. "개인 정보 보호 및 보안" 선택
   3. "확인되지 않은 개발자의 앱 허용" 옵션 활성화
   4. Antigravity 다시 실행
   ```

2. **제어판에서 열기**
   ```
   1. Applications 폴더에서 Antigravity 우클릭
   2. "열기" 선택
   3. 경고 창에서 "열기" 버튼 클릭
   ```

3. **터미널에서 권한 부여**
   ```bash
   sudo xattr -rd com.apple.quarantine /Applications/Antigravity.app
   ```

#### Mac: DMG 파일이 열리지 않음

:::div{.callout}
**증상:**<br>
- DMG 파일을 더블클릭해도 반응이 없음<br>
- 마운트되지 않음
:::

**해결 방법:**
```bash
# 터미널에서 수동 마운트
hdiutil attach ~/Downloads/Antigravity.dmg

# 파일 무결성 확인
hdiutil verify ~/Downloads/Antigravity.dmg
```

#### Windows: 설치 파일이 실행되지 않음

:::div{.callout}
**증상:**<br>
- 설치 파일을 더블클릭해도 아무 반응이 없음<br>
- "이 앱은 PC에서 실행할 수 없습니다" 메시지
:::

**해결 방법:**

1. **관리자 권한으로 실행**
   - 설치 파일 우클릭
   - "관리자 권한으로 실행" 선택

2. **바이러스 백신 확인**
   - 바이러스 백신 소프트웨어에서 차단되었는지 확인
   - 예외 목록에 추가

3. **Windows 업데이트 확인**
   ```
   설정 > Windows 업데이트 > 업데이트 확인
   ```

4. **호환성 모드 실행**
   - 설치 파일 우클릭 > 속성
   - 호환성 탭에서 "호환성 모드에서 이 프로그램 실행" 선택

#### Windows: 설치 중 오류 발생

:::div{.callout}
**증상:**<br>
- 설치 중간에 오류 메시지 표시<br>
- 설치가 중단됨
:::

**해결 방법:**

1. **디스크 공간 확인**
   ```powershell
   Get-PSDrive C | Select-Object Used,Free
   ```

2. **이전 설치 제거**
   ```
   설정 > 앱 > 앱 및 기능
   Antigravity 검색 후 제거
   ```

3. **임시 파일 정리**
   ```powershell
   # 임시 폴더 정리
   Remove-Item $env:TEMP\* -Recurse -Force
   ```

4. **Windows 이벤트 로그 확인**
   ```
   이벤트 뷰어 > Windows 로그 > 응용 프로그램
   오류 항목 확인
   ```

#### Linux: 권한 오류

:::div{.callout}
**증상:**<br>
- "Permission denied" 오류<br>
- 실행 파일이 실행되지 않음
:::

**해결 방법:**
```bash
# 실행 권한 부여
chmod +x antigravity

# 또는 AppImage의 경우
chmod +x Antigravity.AppImage

# 소유자 확인 및 변경
ls -l antigravity
sudo chown $USER:$USER antigravity
```

#### Linux: 의존성 오류

:::div{.callout}
**증상:**<br>
- "libxxx.so not found" 오류<br>
- 필요한 라이브러리가 없다는 메시지
:::

**해결 방법:**

**Debian/Ubuntu:**
```bash
# 의존성 자동 설치
sudo apt-get install -f

# 누락된 라이브러리 확인
ldd $(which antigravity)

# 공통 라이브러리 설치
sudo apt-get install libgtk-3-0 libnotify4 libnss3 libxss1 libxtst6 libatspi2.0-0 libappindicator3-1 libsecret-1-0
```

**Red Hat/Fedora:**
```bash
# 의존성 확인
sudo dnf install -f

# 공통 라이브러리 설치
sudo dnf install gtk3 libnotify nss libXScrnSaver libXtst at-spi2-atk libappindicator-gtk3 libsecret
```

### 실행 문제

#### 프로그램이 시작되지 않음

:::div{.callout}
**증상:**<br>
- 아이콘 클릭해도 반응 없음<br>
- 프로세스가 시작되지 않음
:::

**해결 방법:**

1. **이미 실행 중인 프로세스 확인**
   ```bash
   # Mac/Linux
   ps aux | grep antigravity
   killall Antigravity  # 강제 종료

   # Windows
   tasklist | findstr antigravity
   taskkill /F /IM antigravity.exe
   ```

2. **로그 파일 확인**
   - Mac: `~/Library/Logs/Antigravity/`
   - Windows: `%APPDATA%\Antigravity\logs\`
   - Linux: `~/.config/Antigravity/logs/`

3. **캐시 삭제**
   ```bash
   # Mac
   rm -rf ~/Library/Caches/Antigravity

   # Windows
   Remove-Item $env:APPDATA\Antigravity\Cache -Recurse -Force

   # Linux
   rm -rf ~/.cache/Antigravity
   ```

#### 로그인 문제

:::div{.callout}
**증상:**<br>
- Google 계정 로그인이 되지 않음<br>
- 로그인 화면이 계속 표시됨
:::

**해결 방법:**

1. **인터넷 연결 확인**
   ```bash
   ping -c 3 accounts.google.com
   ```

2. **브라우저 쿠키 확인**
   - 기본 브라우저에서 Google 계정 로그인 상태 확인
   - 쿠키 및 캐시 삭제 후 재시도

3. **방화벽 설정 확인**
   - 포트 443 (HTTPS) 열림 확인
   - Antigravity가 방화벽 예외 목록에 있는지 확인

4. **프록시 설정 확인**
   - 회사나 학교 네트워크 사용 시 프록시 설정 필요할 수 있음
   - 시스템 프록시 설정 확인

#### 성능 문제

:::div{.callout}
**증상:**<br>
- 프로그램이 느리게 실행됨<br>
- 메모리 사용량이 과도함
:::

**해결 방법:**

1. **시스템 리소스 확인**
   ```bash
   # Mac/Linux
   top
   htop  # 설치되어 있는 경우

   # Windows
   작업 관리자 열기 (Ctrl+Shift+Esc)
   ```

2. **다른 프로그램 종료**
   - 불필요한 프로그램 종료하여 리소스 확보

3. **설정 최적화**
   - Antigravity 설정에서 불필요한 기능 비활성화
   - 자동 업데이트 간격 조정

4. **재설치 고려**
   - 문제가 지속되면 완전 제거 후 재설치

### 네트워크 문제

#### 연결 오류

:::div{.callout}
**증상:**<br>
- "연결할 수 없습니다" 오류<br>
- 네트워크 타임아웃
:::

**해결 방법:**

1. **네트워크 연결 확인**
   ```bash
   ping -c 3 8.8.8.8
   ping -c 3 antigravity.google
   ```

2. **DNS 설정 확인**
   ```bash
   # DNS 서버 변경 시도
   # Google DNS: 8.8.8.8, 8.8.4.4
   ```

3. **방화벽 확인**
   - 방화벽에서 Antigravity 허용 확인
   - 포트 443, 80 열림 확인

4. **VPN/프록시 확인**
   - VPN 연결 시 안정성 확인
   - 프록시 설정이 올바른지 확인

## 고급 문제 해결

### 로그 파일 분석

#### Mac
```bash
# 로그 파일 위치
~/Library/Logs/Antigravity/

# 최신 로그 확인
tail -f ~/Library/Logs/Antigravity/main.log

# 시스템 로그 확인
log show --predicate 'process == "Antigravity"' --last 1h
```

#### Windows
```powershell
# 로그 파일 위치
$env:APPDATA\Antigravity\logs\

# 최신 로그 확인
Get-Content $env:APPDATA\Antigravity\logs\main.log -Tail 50

# 이벤트 뷰어 확인
eventvwr.msc
```

#### Linux
```bash
# 로그 파일 위치
~/.config/Antigravity/logs/

# 최신 로그 확인
tail -f ~/.config/Antigravity/logs/main.log

# 시스템 로그 확인
journalctl -u antigravity -f
```

### 완전 제거 및 재설치

#### Mac
```bash
# 앱 제거
rm -rf /Applications/Antigravity.app

# 설정 파일 제거
rm -rf ~/Library/Application\ Support/Antigravity
rm -rf ~/Library/Preferences/com.google.Antigravity.plist
rm -rf ~/Library/Caches/Antigravity
rm -rf ~/Library/Logs/Antigravity

# 재설치
# 공식 사이트에서 다시 다운로드하여 설치
```

#### Windows
```powershell
# 프로그램 제거
winget uninstall Google.Antigravity

# 설정 파일 제거
Remove-Item $env:APPDATA\Antigravity -Recurse -Force
Remove-Item $env:LOCALAPPDATA\Antigravity -Recurse -Force

# 레지스트리 정리 (고급)
# regedit에서 HKEY_CURRENT_USER\Software\Google\Antigravity 삭제
```

#### Linux
```bash
# 패키지 제거
sudo apt-get remove antigravity  # Debian/Ubuntu
sudo dnf remove antigravity      # Fedora

# 설정 파일 제거
rm -rf ~/.config/Antigravity
rm -rf ~/.cache/Antigravity
rm -rf ~/.local/share/Antigravity

# 재설치
# 공식 사이트에서 다시 다운로드하여 설치
```
