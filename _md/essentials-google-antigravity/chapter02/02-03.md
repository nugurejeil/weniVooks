## 개요

:::div{.box}
Google Anti-Gravity가 올바르게 설치되었는지 확인하는 방법을 안내합니다.
:::

## 기본 확인 사항

### 1. 프로그램 실행 확인

#### Mac

- Applications 폴더에서 Antigravity 앱이 보이는지 확인
- 앱을 더블클릭하여 실행 시도

#### Windows

- 바탕화면에 Antigravity 아이콘이 있는지 확인
- 시작 메뉴에서 "Antigravity" 검색 가능한지 확인

#### Linux

- 터미널에서 `antigravity` 명령어 입력
- 애플리케이션 메뉴에서 Antigravity 검색

### 2. 실행 테스트

1. Antigravity를 실행합니다
2. Google 계정 로그인 화면이 나타나는지 확인합니다
3. 로그인 화면이 정상적으로 표시되면 설치 성공입니다

## 상세 확인 방법

### Mac

- Applications 폴더에서 Antigravity 앱 확인
- 터미널: `ls -la /Applications/ | grep -i antigravity`

### Windows

- 설정 > 앱 > 앱 및 기능에서 "Antigravity" 검색
- PowerShell: `Get-AppxPackage | Where-Object {$_.Name -like "*Antigravity*"}`

### Linux

- `which antigravity` (실행 파일 위치)
- `dpkg -l | grep antigravity` (Debian/Ubuntu)
- `rpm -qa | grep antigravity` (Red Hat/Fedora)

## 기능 확인

1. Google 계정으로 로그인 시도
2. 메인 화면이 정상적으로 로드되는지 확인
3. 설정 메뉴 접근 가능한지 확인

네트워크 연결 및 시스템 리소스 확인이 필요하면 ::a[문제 해결 가이드]{class='btn-link' href="02-04"}를 참고하세요.
