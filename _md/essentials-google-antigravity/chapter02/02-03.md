## 개요

:::div{.box}
Google Anti-Gravity가 올바르게 설치되었는지 확인하는 방법을 안내합니다.
:::

## 기본 확인 사항

### 1. 프로그램 실행 확인

#### Mac

- Applications 폴더에서 Antigravity 앱이 보이는지 확인
- 앱을 더블클릭하여 실행 시도

#### Windows

- 바탕화면에 Antigravity 아이콘이 있는지 확인
- 시작 메뉴에서 "Antigravity" 검색 가능한지 확인

![안티그래비티 아이콘](/images/essentials-google-antigravity/02-04-windows-exe3.png)

#### Linux

- 터미널에서 `antigravity` 명령어 입력
- 애플리케이션 메뉴에서 Antigravity 검색

### 2. 실행 테스트

![첫 실행화면](/images/essentials-google-antigravity/02-06-windows-exe4.png)

1. Antigravity를 실행합니다
    - next
    - Start fresh -> next
    - 테마 선택
    - Agent-assisted development(AI에게 도움 받기), 나머지도 기본 값 -> next
    - 기본값 -> next
2. Google 계정 로그인 화면이 나타나는지 확인합니다
3. 로그인 화면이 정상적으로 표시되면 설치 성공입니다

![구글 로그인 화면](/images/essentials-google-antigravity/02-06-windows-exe5.png)
