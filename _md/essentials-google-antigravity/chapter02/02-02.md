## 개요

Google Anti-Gravity는 Mac, Windows, Ubuntu를 지원합니다. 각 운영체제에 맞는 설치 방법을 선택하세요.

![최소 요구사항](/images/essentials-google-antigravity/02-01-minimum.png)

## [공식 다운로드 홈페이지](https://antigravity.google/download)

## Mac 설치

### 시스템 요구사항

- macOS 12 (Monterey) 이상
- Apple Silicon 또는 Intel 프로세서 (X86 미지원)

### 프로세서 확인

1. Apple 메뉴 > "이 Mac에 관하여" 선택
2. "프로세서" 항목에서 확인:
   - **Apple Silicon**: M1/M2/M3 등
   - **Intel**: Intel Core 시리즈

![Mac 프로세서 확인](images/essentials-google-antigravity/02-03-mac-spec.png)

### 설치 단계

**1단계: 다운로드**

- Apple Silicon: "Download for Apple Silicon" 선택
- Intel: "Download for Intel" 선택

**2단계: DMG 파일 마운트**

1. 다운로드 폴더에서 `Antigravity.dmg` 파일 더블클릭
2. Finder 창이 자동으로 열림

**3단계: Applications 폴더로 설치**

1. Antigravity 아이콘을 Applications 폴더로 드래그 앤 드롭
2. 복사 완료 확인
3. DMG 파일 언마운트 (`Cmd + E`)

![Mac 설치 과정](images/02-02-mac-install.png)
*그림 2-3: Mac 설치 과정*

**4단계: 설치 완료**

- Applications 폴더에 Antigravity가 설치되었는지 확인

## Windows 설치

### 시스템 요구사항

- Windows 10 (64비트) 이상

### 시스템 아키텍처 확인

1. `Win + I` > 시스템 > 정보
2. "시스템 종류" 확인:
   - **64비트 운영 체제**: x64 버전
   - **ARM 기반 프로세서**: ARM64 버전

![Windows 시스템 아키텍처 확인](/images/essentials-google-antigravity/02-04-windows-spec.png)

### 설치 단계

**1단계: 다운로드**

- x64 시스템: "Download for x64" 선택
- ARM64 시스템: "Download for ARM64" 선택

**2단계: 설치 프로그램 실행**

1. 다운로드 폴더에서 `AntigravitySetup.exe` 더블클릭

![설치 프로그램](/images/essentials-google-antigravity/02-04-windows-exe.png)

**3단계: 설치 마법사 진행**

1. 환영 화면: "Next" 클릭
2. 라이선스 동의: "I Agree" 선택
3. 설치 경로: 기본 경로 유지 (권장)
4. 설치 옵션: 바탕화면 바로가기, 시작 메뉴 추가, 기타에 있는 모든 체크 박스 체크
5. 설치 시작: "Install" 클릭

![설치 프로그램](/images/essentials-google-antigravity/02-04-windows-exe2.png)

**4단계: 설치 완료**

- "Installation Complete" 메시지 확인
- "Finish" 클릭하여 완료

## Ubuntu 설치

### 시스템 요구사항

- Ubuntu 20.04 이상

### 설치 단계

**1단계: 공식 사이트에서 다운로드**

1. 웹 브라우저를 엽니다
2. [공식 다운로드 사이트 리눅스버전](https://antigravity.google/download/linux) 접속

![리눅스 설치 페이지](/images/essentials-google-antigravity/02-05-linux-exe.png)

**2단계: 터미널 열기**

1. `Ctrl + Alt + T` 키를 눌러 터미널 열기

**3단계: 명령어를 사용한 설치**

```bash
sudo mkdir -p /etc/apt/keyrings
curl -fsSL https://us-central1-apt.pkg.dev/doc/repo-signing-key.gpg | \
  sudo gpg --dearmor -o /etc/apt/keyrings/antigravity-repo-key.gpg
echo "deb [signed-by=/etc/apt/keyrings/antigravity-repo-key.gpg] https://us-central1-apt.pkg.dev/projects/antigravity-auto-updater-dev/ antigravity-debian main" | \
  sudo tee /etc/apt/sources.list.d/antigravity.list > /dev/null
```

![우분투 명령어 설치1](/images/essentials-google-antigravity/02-05-linux-exe2.png)

**비밀번호를 입력하라는 메시지가 나타나면 사용자 비밀번호를 입력합니다.**

```bash
sudo apt update
```

![우분투 명령어 설치2](/images/essentials-google-antigravity/02-05-linux-exe3.png)

```bash
sudo apt install antigravity
```

![우분투 명령어 설치3](/images/essentials-google-antigravity/02-05-linux-exe4.png)

**5단계: 의존성 문제 해결 (필요한 경우)**

설치 중 오류가 발생하면 다음 명령어 실행:

```bash
sudo apt install -f
```

이 명령어는 필요한 의존성을 자동으로 설치합니다.
