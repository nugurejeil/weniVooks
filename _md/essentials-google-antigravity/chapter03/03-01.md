## 운영체제별 실행 방법

### Mac

- **Spotlight 검색**: `Cmd + Space` → "Antigravity" 입력
- **Applications 폴더**: Antigravity 앱 더블클릭
- **Dock**: 앱을 Dock에 고정하여 빠르게 실행

### Windows

![Windows 바탕화면 아이콘](/images/essentials-google-antigravity/02-04-windows-exe3.png)

- **시작 메뉴**: "Antigravity" 검색
- **바탕화면 아이콘**: 더블클릭

### Linux

- **터미널**: `antigravity` 입력
- **애플리케이션 메뉴**: "Antigravity" 검색

## 첫 실행 과정

### 1단계: 시작 화면

Antigravity를 실행하면 시작 화면이 표시됩니다.

![시작 화면](/images/essentials-google-antigravity/02-06-windows-exe4.png)

1. "Next" 클릭
2. "Start fresh" 선택 → "Next"
3. 테마 선택 (Light/Dark)
4. "Agent-assisted development" 선택 → "Next"
5. 기본값 유지 → "Next"

### 2단계: Google 계정 로그인

![Google 로그인 화면](/images/essentials-google-antigravity/02-06-windows-exe5.png)

1. Google 계정 선택 또는 이메일 입력
2. 비밀번호 입력
3. 2단계 인증 설정 시 인증 코드 입력

![Google 로그인 완료 화면](/images/essentials-google-antigravity/02-06-windows-exe6.png)

## 기본 인터페이스

로그인 완료 후 기본 인터페이스가 표시됩니다.

![기본 인터페이스](/images/essentials-google-antigravity/03-01-main-interface.png)

### 주요 구성 요소

| 구성 요소 | 설명 |
|-----------|------|
| **Agent Manager** | AI 에이전트 생성 및 관리 |
| **Editor** | 코드 작성 및 편집 |
| **Browser** | 웹 브라우징 (Chrome Extension 필요) |
| **Command** | 터미널 명령 실행 |
| **Tab** | 여러 작업을 탭으로 관리 |

### 첫 에이전트 생성

![에이전트 생성](/images/essentials-google-antigravity/chapter03/03-01-create-agent.png)

1. Agent Manager 열기 (`Cmd/Ctrl + Shift + A`)
2. "새 에이전트 생성" 클릭
3. 에이전트 이름 입력
4. 에이전트 생성 완료

## 자주 쓰는 단축키

| 단축키 | 기능 |
|--------|------|
| `Cmd/Ctrl + E` | 빠른 액세스 |
| `Cmd/Ctrl + Tab` | 탭 전환 |
| `Cmd/Ctrl + Shift + A` | Agent Manager 열기 |
| `Cmd/Ctrl + L` | AI 채팅 열기 |
| `Cmd/Ctrl + ,` | 설정 열기 |
