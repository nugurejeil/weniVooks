## 처음 외울 단축키 TOP 5

Antigravity를 처음 사용한다면 이 핵심 단축키 5개만 먼저 외우시는것을 추천드립니다.

| 우선순위 | 기능 | Mac | Windows/Linux | 설명 |
|:--------:|------|-----|---------------|------|
| 1 | **모드 전환** | `Cmd + E` | `Ctrl + E` | 에디터 ↔ 에이전트 모드 전환 |
| 2 | **저장** | `Cmd + S` | `Ctrl + S` | 현재 파일 저장 |
| 3 | **AI 채팅** | `Cmd + L` | `Ctrl + L` | AI와 대화 시작 |
| 4 | **명령 팔레트** | `Cmd + Shift + P` | `Ctrl + Shift + P` | 모든 명령 검색 |
| 5 | **설정** | `Cmd + ,` | `Ctrl + ,` | 설정 화면 열기 |

:::div{.callout}
**핵심은 `Cmd/Ctrl + E`**

에디터 모드와 에이전트 모드를 자유롭게 오가는 것이 Antigravity 활용의 핵심입니다.
:::

## 에이전트 모드 단축키

에이전트와 협업할 때 사용하는 단축키입니다.

| 기능 | Mac | Windows/Linux | 설명 |
|------|-----|---------------|------|
| 모드 전환 | `Cmd + E` | `Ctrl + E` | 에디터 ↔ 에이전트 모드 |
| AI 채팅 | `Cmd + L` | `Ctrl + L` | 사이드 패널에서 AI와 대화 |
| 인라인 편집 | `Cmd + I` | `Ctrl + I` | 코드 블록 선택 후 AI에게 수정 요청 |

### 인라인 편집 사용 예시

1. 수정하고 싶은 코드 블록 선택
2. `Cmd/Ctrl + I` 입력
3. "이 함수를 async로 변경해줘" 등 요청 입력
4. AI가 제안한 변경사항 확인 후 적용

## 파일 작업

| 기능 | Mac | Windows/Linux |
|------|-----|---------------|
| 새 파일 | `Cmd + N` | `Ctrl + N` |
| 파일 열기 | `Cmd + O` | `Ctrl + O` |
| 저장 | `Cmd + S` | `Ctrl + S` |
| 빠른 파일 열기 | `Cmd + P` | `Ctrl + P` |
| 모든 파일에서 검색 | `Cmd + Shift + F` | `Ctrl + Shift + F` |

:::div{.callout}
**빠른 파일 열기 팁**

`Cmd/Ctrl + P`를 누르고 파일명 일부만 입력해도 됩니다. 예: "index" 입력 → index.html, index.js 등 표시
:::

## 탭 관리

| 기능 | Mac | Windows/Linux |
|------|-----|---------------|
| 탭 닫기 | `Cmd + W` | `Ctrl + W` |
| 다음 탭 | `Ctrl + Tab` | `Ctrl + Tab` |
| 이전 탭 | `Ctrl + Shift + Tab` | `Ctrl + Shift + Tab` |
| 닫은 탭 다시 열기 | `Cmd + Shift + T` | `Ctrl + Shift + T` |

## 편집

| 기능 | Mac | Windows/Linux |
|------|-----|---------------|
| 실행 취소 | `Cmd + Z` | `Ctrl + Z` |
| 다시 실행 | `Cmd + Shift + Z` | `Ctrl + Y` |
| 찾기 | `Cmd + F` | `Ctrl + F` |
| 바꾸기 | `Cmd + Option + F` | `Ctrl + H` |
| 현재 줄 삭제 | `Cmd + Shift + K` | `Ctrl + Shift + K` |
| 줄 복제 | `Cmd + Shift + D` | `Ctrl + Shift + D` |
| 줄 이동 (위) | `Option + ↑` | `Alt + ↑` |
| 줄 이동 (아래) | `Option + ↓` | `Alt + ↓` |

## 화면 제어

| 기능 | Mac | Windows/Linux |
|------|-----|---------------|
| 사이드바 토글 | `Cmd + B` | `Ctrl + B` |
| 터미널 토글 | `` Cmd + ` `` | `` Ctrl + ` `` |
| 확대 | `Cmd + =` | `Ctrl + =` |
| 축소 | `Cmd + -` | `Ctrl + -` |
| 전체 화면 | `Cmd + Ctrl + F` | `F11` |
