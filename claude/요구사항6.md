수정한 것을 보니까 챕터의 각 `.md`파일 마다 `# 1. 제목` 이렇게 시작해야 하는데 `# 1.3` 등으로 소제목으로 시작해. 내가 원한 것은 아래와 같은 양식이야.

- `chapter01` 폴더에 `01-2.md`는 `# 1.2 위니브 회사 Claude Code 바이브 코딩 도입기` 이렇게 시작하는 것이 아니라 `# 1. 위니브 회사 Claude Code 바이브 코딩 도입기` 이렇게 시작해야해.

그리고 이미지 폴더명이 각각의 챕터명으로 되어 있지 않아. 예를 들어 챕터 1의 이미지들은 `essentials-vibecoding/chapter1` 이런 식으로 그 위에 챕터명 폴더명이 있어야 나중에 유지보수하기 좋을 것 같아. 아래 작업을 수행해줘.

1. 이미지 폴더 수정
2. 각 마크다운 파일에 있는 이미지 경로 수정
3. 경로명에 공백과 한글이 있으면 이미지가 안떠. 그래서 이미지 폴더명, 파일명 모두 한글이나 공백이 없도록 해줘.
