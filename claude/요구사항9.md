'pdf_export'라는 폴더에 원하는 책을 PDF로 만들어주는 파이썬 파일이 있어 아래 문제를 해결해줘.

1. 첫 페이지에는 표지만 있었으면 좋겠어. 표지 이미지가 있으니 그걸로 첫 페이지를 만들어줘. 그 다음은 목차가 나오면 돼.
2. 엥커 태그는 아래와 같이 그대로 텍스트로 나와버려. 이걸 이름과 링크 형식으로 바꿔줘. 독자가 보기에 편하게. PDF 독자는 클릭을 할 수 없다는 사실을 고려해야해.
2.1 지금 나오고 있는 방식
```
::a[노션 요금제]{class='btn-link' href="https://www.notion.so/ko-kr/pricing" target="_blank"}
```

2.2 PDF로 출력이 되어야 하는 방식
```
노션 요금제: https://www.notion.so/ko-kr/pricing
```

3. 그리고 일부 이미지는 아래와 같은 형식으로 나오고 있어. 이걸 이미지로 제대로 나오게 해줘.
```
image.png
```

이렇게 나오는 이미지의 공통점은 %가 있다는 것이야. 예를 들어 아래 이미지는 위에 이미지는 잘 나오는데 아래 이미지는 잘 나오지 않아.

```
![image.png](/images/essentials-notion/2_4_image.png)

수신된 데이터는 `수신함`에서 볼 수 있습니다.

![image.png](/images/essentials-notion/2_4_image%202.png)
```

4. 머메이드 블록으로 된 것은 머메이드로 나와야 하는데 텍스트로만 나와. 머메이드 블록은 가능하면 이미지로 넣어줘.
