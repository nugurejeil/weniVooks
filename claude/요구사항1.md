나는 React에 대한 교안을 집필하고 있어. 이 집필은 React 베이스캠프라는 이름으로 이제 막 React에 입문한 사람들 대상이야. basecamp-react라는 교안을 작성해줘.

# 서비스와 목적
위니북스 서비스는 마크다운 형식의 교안을 편히 읽고 실행할 수 있는 기능을 제공해. HTML, CSS, JS, 머메이드 등을 제공하지만 React는 동작하지 않아. 

이 교안은 기존에 있던 리엑트 바로 시작하기 라는 교안을 기반으로 더 쉽고 간결하게 작성했으면 좋겠어. 기존에 있던 교안은 바로 적용해볼 사람들을 위한 교안이었고, 이 교안은 리엑트에 대한 기초부터 차근차근 배워볼 사람들을 위한 교안이야.

# 작성해야 하는 폴더 구조

아래 폴더 외에는 가급적 건드리지 않았으면 좋겠어. 건드리더라도 나에게 허락을 받아야해.

```
교안: wenivooks/_md/basecamp-react
이미지: wenivooks/public/images
```

# 이전 교안

이전 교안에는 이미지와 코드가 포함되어 있는데 가능하면 충분히 참고해줘.

```
교안: claudedev/react
수업중 코드: claudedev/react-lecture
```
